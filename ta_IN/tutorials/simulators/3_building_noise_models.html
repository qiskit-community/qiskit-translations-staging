சத்தம் மாதிரிகளை உருவாக்குதல்¶

அறிமுகம்¶

சத்தமில்லாத உருவகப்படுத்துதல்களுக்கு தனிப்பயன் இரைச்சல் மாதிரிகளை உருவாக்க Qiskitசத்தம் ஏர் Qiskit தொகுதியை எவ்வாறு பயன்படுத்துவது என்பதை இந்த நோட்புக் அறிமுகப்படுத்துகிறது.

import numpy as np

Qiskit ஏர் சத்தம் தொகுதி¶

உருவகப்படுத்துதலுக்கான தனிப்பயனாக்கப்பட்ட இரைச்சல் மாதிரிகளை உருவாக்க Qiskit ஏர் noise தொகுதி பைதான் வகுப்புகளைக் கொண்டுள்ளது. மூன்று முக்கிய வகுப்புகள் உள்ளன:

  1. இந்த NoiseModel வகுப்பு சத்தம் உருவகப்படுத்துதலுக்குப் பயன்படுத்தப்படும் சத்தம் மாதிரியைச் சேமிக்கும்.

  2. சிபிடிபி கேட் பிழைகளை விவரிக்கும் QuantumError வகுப்பு. இவற்றைப் பயன்படுத்தலாம்:

    • gate அல்லது reset கட்டளைகளுக்குப் பிறகு

    • measure கட்டளைகளுக்கு முன்.

  3. இந்த ReadoutError வகுப்பு கிளாசிக்கல் ரீட்அவுட் பிழைகளை விவரிக்கும்.

குவாண்டம் பிழைகள்¶

QuantumError ஆப்ஜெடை நேரடியாகக் கையாள்வதற்குப் பதிலாக, ஒரு குறிப்பிட்ட வகை அளவுரு குவாண்டம் பிழையை தானாக உருவாக்க பல உதவி செயல்பாடுகள் உள்ளன. இவை `` noise`` தொகுதியில் உள்ளன மற்றும் குவாண்டம் கம்ப்யூட்டிங் ஆராய்ச்சியில் பயன்படுத்தப்படும் பல பொதுவான பிழைகள் வகைகளுக்கான செயல்பாடுகளையும் உள்ளடக்கியது. இந்த செயல்பாட்டு பெயர்கள் மற்றும் அவை தரும் பிழையின் வகை:

நிலையான பிழை செயல்கூறு

விவரங்கள்

kraus_error

ஒரு பொதுவான n-qubit CPTP பிழை சேனல் க்ராஸ் மெட்ரிக்ஸின் பட்டியலாக வழங்கப்பட்டுள்ளது \([K_0, ...]\).

mixed_unitary_error

ஒரு n-qubit கலப்பு யூனிடேரி பிழை யூனிடேரி மேட்ரிக்ஸ் மற்றும் நிகழ்தகவுகளின் பட்டியலாக கொடுக்கப்பட்டுள்ளது \([(U_0, p_0),...]\).

coherent_unitary_error

ஒரு n-qubit coherent யூனிடேரி பிழை ஒரு ஒரே யூனிடேரி மேட்ரிக்ஸ் \(U\) என்று கொடுக்கப்பட்டுள்ளது.

pauli_error

ஒரு n-qubit பவுலி பிழை சேனல் (கலப்பு யூனிடேரி) பவுலியின் மற்றும் நிகழ்தகவுகளின் பட்டியலாக வழங்கப்பட்டுள்ளது \([(P_0, p_0),...]\)

depolarizing_error

ஒரு n-qubit டிப்போலரைசிங் பிழை சேனல் டிப்போலரைசேஷன் நிகழ்தகவு மூலம் அளவுருவாக்கப்பட்டுள்ளது \(p\).

``reset_error` `

ஒற்றை-கியூபிட் மீட்டமைப்பு பிழை நிகழ்தகவுகளால் அளவுருவாக்கப்பட்ட \(p_0, p_1\) மீட்டமைத்தல் \(|0\rangle\), \(|1\rangle\) நிலை.

` ` thermal_relaxation_error ` `

ஒற்றை குவிட் வெப்ப தளர்வு சேனல் தளர்வு நேர மாறிலிகளால் அளவுருவாக்கப்பட்ட \(T_1\), \(T_2\), கேட் நேரம் \(t\), மற்றும் வெப்ப குடியேற்ற கிளர்வு நிலை \(p_1\).

phase_amplitude_damping_error

ஒரு ஒற்றை-குவிட் பொதுப்படையான ஒருங்கிணைந்த மாற்று மின்னோட்ட இயக்கப் படிநிலை மற்றும் அலைவீச்சு தணிக்கும் பிழை சேனல் அலைவீச்சு தணிக்கும் அளவுரு \(\lambda\) , ஒரு மாற்று மின்னோட்ட இயக்கப் படிநிலை தணிக்கும் அளவுரு \(\gamma\), மற்றும் ஒரு வெப்ப குடியேற்ற கிளர்வு நிலை \(p_1\) ஆல் வழங்கப்பட்டது.

amplitude_damping_error

ஒற்றை-குவிட் பொதுமைப்படுத்தப்பட்ட அலைவீச்சு தணிக்கும் பிழை சேனல் ஆனது ஒரு வீச்சு தணிக்கும் அளவுரு \(\lambda\), மற்றும் ஒரு வெப்ப குடியேற்ற கிளர்வு நிலை \(p_1\) ஆல் வழங்கப்பட்டது.

phase_damping_error

ஒற்றை-குவிட் பொதுமைப்படுத்தப்பட்ட அலைவீச்சு தணிக்கும் பிழை சேனல் ஆனது ஒரு வீச்சு தணிக்கும் அளவுரு \(\gamma\) ஆல் வழங்கப்பட்டது

குவாண்ட் பிழைகளை ஒருங்கிணைத்தல்¶

QuantumError நிகழ்வுகளை, கலவை, டென்சர் தயாரிப்பு மற்றும் டென்சர் விரிவாக்கம் (தலைகீழ் ஆர்டர் டென்சர் தயாரிப்பு) ஆகியவற்றை இணைந்து புதிய QuantumError உருவாக்க:

  • கலவை: \(\cal{E}(\rho)=\cal{E_2}(\cal{E_1}(\rho))\) அப்படியே error = error1.compose(error2)

  • டென்சர் தயாரிப்பு: \(\cal{E}(\rho) =(\cal{E_1}\otimes\cal{E_2})(\rho)\) அப்படியே error error1.tensor(error2)

  • தயாரிப்பு விரிவாக்கு: \(\cal{E}(\rho) =(\cal{E_2}\otimes\cal{E_1})(\rho)\) என error error1.expand(error2)

வாசிப்பு பிழை¶

கிளாசிக்கல் ரீட்அவுட் பிழைகள் ஆனது ஒதுக்கீட்டு நிகழ்தகவு வெக்ட்டார்களின் பட்டியல் \(P(A|B)\): ஆல் குறிப்பிடப்படுகின்றன:

  • \(A\) என்பது recorded கிளாசிக்கல் பிட் மதிப்பு

  • \(B\) என்பது அளவீட்டிலிருந்து திரும்பிய true பிட் மதிப்பு

எ.கா. 1 குவிட்டுகளுக்கு: $ P(A|B) = [P(A|0), P(A|1)]$.

சத்தம் மாதிரியில் பிழைகளைச் சேர்த்தல்¶

இரைச்சல் மாதிரியில் ஒரு குவாண்டம் பிழையைச் சேர்க்கும்போது, அது செயல்படும் * அறிவுறுத்தல் * வகையையும், அதைப் பயன்படுத்துவதற்கு என்ன வினவல்களையும் குறிப்பிட வேண்டும். குவாண்டம் பிழைகளுக்கு மூன்று வழக்குகள் உள்ளன:

  1. அனைத்து-விகிதமான குவாண்ட் பிழை

  2. குறிப்பிட்ட குறும்பட குவாண்ட் பிழை

  3. உள்ளூர் அல்லாத குவாண்டம் பிழை

அனைத்து-விகிதமான குவாண்ட் பிழை¶

ஒரு அறிவுறுத்தலின் எந்தவொரு நிகழ்விற்கும் இது அதே பிழையைப் பொருத்துகிறது, இது எந்த வினாடிகளில் செயல்படுகிறது என்பதைப் பொருட்படுத்தாமல்.

இது noise_model.add_all_qubit_quantum_error(error, instructions):

[9]:

# Create an empty noise model
noise_model = NoiseModel()

# Add depolarizing error to all single qubit u1, u2, u3 gates
error = depolarizing_error(0.05, 1)
noise_model.add_all_qubit_quantum_error(error, ['u1', 'u2', 'u3'])

# Print noise model info
print(noise_model)

  All-qubits errors: ['u1', 'u2', 'u3']

குறிப்பிட்ட குறும்பட குவாண்ட் பிழை¶

குறிப்பிட்ட வினாடிகளின் பட்டியலில் செயல்படும் அறிவுறுத்தலின் ஏதேனும் நிகழ்வுக்கு இது பிழையைப் பொருத்துகிறது. குவிட்டின் வரிசை முக்கியமானது என்பதை நினைவில் கொள்க: 2-குவிட் வாயிலுக்கு குவிட்ஸ் [0, 1] இல் பயன்படுத்தப்படும் பிழை, எடுத்துக்காட்டாக, குவிட்ஸ் [1, 0] க்கு பயன்படுத்தப்படும் ஒன்றுக்கு வேறுபட்டது.

இது noise_model.add_quantum_error(error, instructions, qubits):

[10]:

# Create an empty noise model
noise_model = NoiseModel()

# Add depolarizing error to all single qubit u1, u2, u3 gates on qubit 0 only
error = depolarizing_error(0.05, 1)
noise_model.add_quantum_error(error, ['u1', 'u2', 'u3'], [0])

# Print noise model info
print(noise_model)

உள்ளூர் அல்லாத குவிட் குவாண்டம் பிழை¶

ஒரு குறிப்பிட்ட கேட் குவிட்களில் செயல்படும் ஒரு அறிவுறுத்தலின் ஏதேனும் நிகழ்வுக்குப் பிறகு இது ஒரு குறிப்பிட்ட சத்தம் குவிட்டுகளுக்கு ஒரு பிழையைப் பொருத்துகிறது.

இது noise_model.add_quantum_error(error, instructions, instr_qubits, error_qubits) என்று சேர்க்கப்பட்டுள்ளது:

noise_model.add_nonlocal_quantum_error(error, ['u1', 'u2', 'u3'], [0], [2])

# Print noise model info
print(noise_model)

சத்தம் மாதிரியுடன் சத்தமில்லாத உருவகப்படுத்துதலை செயல்படுத்துகிறது¶

AerSimulator(noise_model=noise_model) கட்டளை கொடுக்கப்பட்ட இரைச்சல் மாதிரியில் கட்டமைக்கப்பட்ட ஒரு சிமுலேட்டரை வழங்குகிறது. சிமுலேட்டரின் இரைச்சல் மாதிரியை அமைப்பதோடு மட்டுமல்லாமல், இரைச்சல் மாதிரியின் வாயில்களின்படி, இது சிமுலேட்டரின் அடிப்படை வாயில்களையும் மீறுகிறது.

சத்தம் மாதிரி எடுத்துக்காட்டுகள்¶

இரைச்சல் மாதிரிகளின் சில எடுத்துக்காட்டுகளை இப்போது தருவோம். எங்கள் ஆர்ப்பாட்டங்களுக்கு, n-qubit GHZ நிலையை உருவாக்கும் எளிய சோதனை சுற்றுகளைப் பயன்படுத்துவோம்:

சத்தம் எடுத்துக்காட்டு 1: அடிப்படை பிட்-ஃபிளிப் பிழை சத்தம் மாதிரி¶

குவாண்டம் தகவல் கோட்பாடு ஆராய்ச்சியில் பொதுவான ஒரு எளிய பொம்மை சத்தம் மாதிரி உதாரணத்தைக் கருத்தில் கொள்வோம்:

  • ஒற்றை குவிட் வாயிலைப் பயன்படுத்தும்போது, `` p_gate1`` நிகழ்தகவுடன் குவிட்டின் நிலையை புரட்டவும்.

  • 2-குவிட் வாயிலைப் பயன்படுத்தும்போது ஒவ்வொரு குவிட்டிற்கும் ஒற்றை-குவிட் பிழைகளைப் பயன்படுத்துங்கள்.

  • நிகழ்தகவு `` p_reset`` உடன் 0 க்கு பதிலாக 1 க்கு ஒரு குவிட் மீட்டமைப்பை மீட்டமைக்கும்போது.

  • ஒரு குவிட்டை அளவிடும்போது, `` p_meas`` நிகழ்தகவுடன் குவிட்டின் நிலையை புரட்டவும்.

[14]:

# Example error probabilities
p_reset = 0.03
p_meas = 0.1
p_gate1 = 0.05

# QuantumError objects
error_reset = pauli_error([('X', p_reset), ('I', 1 - p_reset)])
error_meas = pauli_error([('X',p_meas), ('I', 1 - p_meas)])
error_gate1 = pauli_error([('X',p_gate1), ('I', 1 - p_gate1)])
error_gate2 = error_gate1.tensor(error_gate1)

# Add errors to noise model
noise_bit_flip = NoiseModel()
noise_bit_flip.add_all_qubit_quantum_error(error_reset, "reset")
noise_bit_flip.add_all_qubit_quantum_error(error_meas, "measure")
noise_bit_flip.add_all_qubit_quantum_error(error_gate1, ["u1", "u2", "u3"])
noise_bit_flip.add_all_qubit_quantum_error(error_gate2, ["cx"])

print(noise_bit_flip)

எடுத்துக்காட்டு 2: T1/T2 வெப்ப தளர்வு¶

இப்போது குவிட் சூழலுடன் வெப்ப தளர்த்தலை அடிப்படையாகக் கொண்ட மிகவும் யதார்த்தமான பிழை மாதிரியைக் கவனியுங்கள்: * ஒவ்வொரு குவிட்டும் வெப்ப தளர்வு நேர மாறிலி மூலம் அளவுருவாக்கப்படுகிறது \(T_1\) மற்றும் ஒரு குறைக்கும் நேர மாறிலி :math:` T_2`. * நம்மிடம் இருக்க வேண்டும் என்பதை நினைவில் கொள்க \(T_2 \ le 2 T_1\). Instructions * அறிவுறுத்தல்களின் பிழை விகிதங்கள் கேட் நேரங்கள் மற்றும் குவிட் ஆகியவற்றால் தீர்மானிக்கப்படுகின்றன \(T_1\) ,:math:` T_2` மதிப்புகள்.

[16]:

# T1 and T2 values for qubits 0-3
T1s = np.random.normal(50e3, 10e3, 4) # Sampled from normal distribution mean 50 microsec
T2s = np.random.normal(70e3, 10e3, 4)  # Sampled from normal distribution mean 50 microsec

# Truncate random T2s <= T1s
T2s = np.array([min(T2s[j], 2 * T1s[j]) for j in range(4)])

# Instruction times (in nanoseconds)
time_u1 = 0   # virtual gate
time_u2 = 50  # (single X90 pulse)
time_u3 = 100 # (two X90 pulses)
time_cx = 300
time_reset = 1000  # 1 microsecond
time_measure = 1000 # 1 microsecond

# QuantumError objects
errors_reset = [thermal_relaxation_error(t1, t2, time_reset)
                for t1, t2 in zip(T1s, T2s)]
errors_measure = [thermal_relaxation_error(t1, t2, time_measure)
                  for t1, t2 in zip(T1s, T2s)]
errors_u1  = [thermal_relaxation_error(t1, t2, time_u1)
              for t1, t2 in zip(T1s, T2s)]
errors_u2  = [thermal_relaxation_error(t1, t2, time_u2)
              for t1, t2 in zip(T1s, T2s)]
errors_u3  = [thermal_relaxation_error(t1, t2, time_u3)
              for t1, t2 in zip(T1s, T2s)]
errors_cx = [[thermal_relaxation_error(t1a, t2a, time_cx).expand(
             thermal_relaxation_error(t1b, t2b, time_cx))
              for t1a, t2a in zip(T1s, T2s)]
               for t1b, t2b in zip(T1s, T2s)]

# Add errors to noise model
noise_thermal = NoiseModel()
for j in range(4):
    noise_thermal.add_quantum_error(errors_reset[j], "reset", [j])
    noise_thermal.add_quantum_error(errors_measure[j], "measure", [j])
    noise_thermal.add_quantum_error(errors_u1[j], "u1", [j])
    noise_thermal.add_quantum_error(errors_u2[j], "u2", [j])
    noise_thermal.add_quantum_error(errors_u3[j], "u3", [j])
    for k in range(4):
        noise_thermal.add_quantum_error(errors_cx[j][k], "cx", [j, k])

print(noise_thermal)
