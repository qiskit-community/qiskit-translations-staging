Qiskit Runtime ジョブの最大実行時間はどれくらいですか？¶

公平性を確保するために、Qiskit Runtime ジョブごとに最大実行時間があります。 この制限時間を超えると、強制的にキャンセルされます。 これは、ジョブステータスで Cancelled - Ran too long （キャンセル済み - 実行が長すぎます）と表示されます。 最大実行時間は、1）システム制限 と 2）プログラムで定義された max_execution_time の小さい方です。 システム制限は次のように定義されています。

IBM Cloud上の Qiskit Runtime¶

ジョブ実行時間のシステムしきい値は、シミュレーターで実行されるジョブの場合は 3 時間、物理システムで実行されているジョブの場合は 8 時間です。

その他の制限事項¶

  • プログラムのサイズは 750KB を超えてはなりません

  • ジョブへの入力のサイズは64MBを超えることはできません。
