Электронная структура¶

Введение¶

Гамильтониан молекулы может быть записан в виде

\[\mathcal{H} = - \sum_I \frac{\nabla_{R_I}^2}{M_I} - \sum_i \frac{\nabla_{r_i}^2}{m_e} - \sum_I\sum_i \frac{Z_I e^2}{|R_I-r_i|} + \sum_i \sum_{j>i} \frac{e^2}{|r_i-r_j|} + \sum_I\sum_{J>I} \frac{Z_I Z_J e^2}{|R_I-R_J|}\]

Поскольку атомные ядра намного тяжелее электронов, характерные времена изменения их состояний сильно различаются, и можно рассматривать поведение этих систем раздельно. Данное приближение называется приближением Борна — Оппенгеймера (адиабатическим приближением).

Как следствие, сначала можно решить задачу, используя координаты ядер только в качестве параметров. А энергетические уровни электронов могут быть найдены путем решения нерелятивистского стационарного уравнения Шредингера,

\[\mathcal{H}_{\text{el}} |\Psi_{n}\rangle = E_{n} |\Psi_{n}\rangle\]

где

\[\mathcal{H}_{\text{el}} = - \sum_i \frac{\nabla_{r_i}^2}{m_e} - \sum_I\sum_i \frac{Z_I e^2}{|R_I-r_i|} + \sum_i \sum_{j>i} \frac{e^2}{|r_i-r_j|}.\]

В частности, энергия основного состояния может быть записана как:

\[E_0 = \frac{\langle \Psi_0 | H_{\text{el}} | \Psi_0 \rangle}{\langle \Psi_0 | \Psi_0 \rangle}\]

где \(\Psi_0\) — волновая функция основного состояния системы.

Однако размерность этой задачи возрастает экспоненциально с количеством степеней свободы. Для решения этой проблемы на квантовом компьютере необходимо подготовить \(\Psi_0\) и затем непосредственно измерить ожидаемое значения гамильтониана (\(E_0\)).

Как же это сделать?

Начальное состояние Хартри — Фока¶

Хорошим отправным пунктом для решения нашей задачи является метод Хартри — Фока (Hartree-Fock, HF). Это метод сведения многочастичной задачи к одночастичной в предположении, что каждая частица двигается в некотором усреднённом самосогласованном поле, создаваемом всеми остальными частицами системы. Он достаточно эффективен и приводит к точному значению обменной энергии, хотя и не учитывает корреляционную энергию электронов. Поэтому обычно начинают с решения задачи в приближении Хартри — Фока, а затем добавляют корреляционные взаимодействия.

После этого гамильтониан может быть представлен в базисе собственных функций HF метода, которые называются молекулярными орбиталями (MO):

\[\hat{H}_{elec}=\sum_{pq} h_{pq} \hat{a}^{\dagger}_p \hat{a}_q + \frac{1}{2} \sum_{pqrs} h_{pqrs} \hat{a}^{\dagger}_p \hat{a}^{\dagger}_q \hat{a}_r \hat{a}_s\]

с одночастичными

\[h_{pq} = \int \phi^*_p(r) \left( -\frac{1}{2} \nabla^2 - \sum_{I} \frac{Z_I}{R_I- r} \right) \phi_q(r)dr\]

и двухчастичными интегралами

\[h_{pqrs} = \int \frac{\phi^*_p(r_1) \phi^*_q(r_2) \phi_r(r_2) \phi_s(r_1)}{|r_1-r_2|}dr_1dr_2.\]

MO (\(\phi_u\)) могут быть заполненными или виртуальными (свободными). Одна MO может содержать два электрона. Однако нужно уточнить, что далее мы будем рассматривать так называемые спин-орбитали, которые связаны со спином электрона, находящимся в состоянии вверх (\(\alpha\)) или вниз (\(\beta\)). Поэтому спин-орбиталь может либо содержать один электрон, либо оставаться свободной.

Посмотрим, как реализовать эти шаги в Qiskit.

Qiskit предоставляет интерфейс к различным классическим программам для решения задачи HF. На настоящий момент это:

  • Gaussian

  • Psi4

  • PyQuante

  • PySCF

В следующем примере мы используем PySCF драйвер для описания нейтральной синглетной молекулы водорода в состоянии равновесной геометрии (с межъядерным расстоянием 0,735 ангстрем).

[1]:

from qiskit_nature.drivers import UnitsType, Molecule
from qiskit_nature.drivers.second_quantization import (
    ElectronicStructureDriverType,
    ElectronicStructureMoleculeDriver,
)

molecule = Molecule(
    geometry=[["H", [0.0, 0.0, 0.0]], ["H", [0.0, 0.0, 0.735]]], charge=0, multiplicity=1
)
driver = ElectronicStructureMoleculeDriver(
    molecule, basis="sto3g", driver_type=ElectronicStructureDriverType.PYSCF
)

Отображение фермионов на кубиты¶

Приведенный в предыдущем разделе гамильтониан записан с помощью фермионных операторов. Для преобразования задачи в вид, подходящий для квантового компьютера, они должны быть отображены в виде спиновых операторов (действительно, кубиты подчиняются спиновой статистике).

Существуют разные варианты такого преобразования, обладающие различными свойствами. Сейчас Qiskit поддерживает следующие отображения:

  • Джордана — Вигнера, Jordan-Wigner (Zeitschrift für Physik, 47, 631-651 (1928))

  • четности, Parity (The Journal of chemical physics, 137(22), 224109 (2012))

  • Бравого — Китаева, Bravyi-Kitaev (Annals of Physics, 298(1), 210-226 (2002))

Особенно интересно отображение Jordan-Wigner, так как оно связывает каждую спин-орбиталь с кубитом (как показано на рисунке выше).

Далее показано, как сформулировать расчет электронной структуры для создания оператора вторичного квантования и преобразователя кубита, которые превратяться в оператор, применяемый к кубиту.

[2]:

from qiskit_nature.problems.second_quantization import ElectronicStructureProblem
from qiskit_nature.converters.second_quantization import QubitConverter
from qiskit_nature.mappers.second_quantization import JordanWignerMapper, ParityMapper

[5]:

qubit_converter = QubitConverter(mapper=ParityMapper(), two_qubit_reduction=True)
qubit_op = qubit_converter.convert(second_q_op[0], num_particles=es_problem.num_particles)
print(qubit_op)
