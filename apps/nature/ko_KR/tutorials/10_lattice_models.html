격자 모형들¶

소개¶

양자물리학에서 (응집물질물리학과 고에너지 물리학을 포함) 우리는 종종 격자 모형들을 연구한다. 예를 들어, 고체 내에서 전자의 움직임을 생각해보면, 우리는 원자들의 위치들이 격자점으로 정의된 격자 상에서 전자의 운동을 연구할 수 있다. 본 튜토리얼에서는 Lattice 클래스를 활용하여 LineLattice`, SquareLattice, HyperCubicLattice, TriangularLattice, 그리고 일반적인 격자 등 다양한 격자 시스템을 생성할 수 있는 방법을 제시한다. 또한, 격자 모형의 하나인 페르미-허바드 모형 (Fermi-Hubbard model) 의 예시 또한 제시한다. 우리는 FermiHubbardModel 클래스를 활용하여 주어진 격자에 대한 페르미-허바드 모형의 해밀토니안 (Hamiltonian) 을 어떻게 정의할 수 있는지 다룰 것이다.

[1]:

from math import pi

import numpy as np
import retworkx as rx
from qiskit_nature.problems.second_quantization.lattice import (
    BoundaryCondition,
    FermiHubbardModel,
    HyperCubicLattice,
    Lattice,
    LatticeDrawStyle,
    LineLattice,
    SquareLattice,
    TriangularLattice,
)

LineLattice¶

LineLattice 클래스는 1차원 격자를 제공한다. 우리는 다음과 같이 1차원 격자를 구성할 수 있다.

[2]:

num_nodes = 11
boundary_condition = BoundaryCondition.OPEN
line_lattice = LineLattice(num_nodes=num_nodes, boundary_condition=boundary_condition)

격자를 시각화하면 다음과 같다.

[3]:

line_lattice.draw()

우리는 boundary_condition 인자를 BoundaryCondition.PERIODIC 로 명시함으로써 주기적 경계 조건을 갖는 1차원 격자를 구성할 수도 있다.

[4]:

num_nodes = 11
boundary_condition = BoundaryCondition.PERIODIC
line_lattice = LineLattice(num_nodes=num_nodes, boundary_condition=boundary_condition)

line_lattice.draw()

경계조건을 무시하고 격자를 그리고 싶을 때에는, draw_without_boundary 메소드를 사용한다.

[5]:

line_lattice.draw_without_boundary()

우리는 격자 상의 모서리들에 실수 혹은 복소수 가중치를 정의할 수 있다. 이는 edge_parameter 인자에 값을 전달함으로써 가능하다. 또한, onsite_parameter 인자에 값을 전달함으로써 자가 루프 (self-loop) 들의 가중치도 정의할 수 있다.

[6]:

num_nodes = 11
boundary_condition = BoundaryCondition.PERIODIC
edge_parameter = 1.0 + 1.0j
onsite_parameter = 1.0
line_lattice = LineLattice(
    num_nodes=num_nodes,
    edge_parameter=edge_parameter,
    onsite_parameter=onsite_parameter,
    boundary_condition=boundary_condition,
)
set(line_lattice.graph.weighted_edge_list())

SquareLattice¶

``SquareLattice``는 2차원 격자를 제공한다. 여기서 우리는 열린 경계 조건을 갖는 2차원 격자를 구성한다.

[9]:

rows = 5
cols = 4
boundary_condition = BoundaryCondition.OPEN
square_lattice = SquareLattice(rows=rows, cols=cols, boundary_condition=boundary_condition)

square_lattice.draw()

우리는 boundary_condition 인자를 튜플로 설정함으로써, 각 방향에 대한 경계조건을 명시할 수 있다.

[10]:

rows = 5
cols = 4
boundary_condition = (
    BoundaryCondition.OPEN,
    BoundaryCondition.PERIODIC,
)  # open in the x-direction, periodic in the y-direction
square_lattice = SquareLattice(rows=rows, cols=cols, boundary_condition=boundary_condition)

square_lattice.draw()

1차원 격자에서처럼, 우리는 격자 상의 모서리들과 자가 루프들에 가중치를 줄 수 있다. 여기서는, 튜플을 통해 각 방향에 대한 가중치를 주는 것이 가능하다.

[11]:

rows = 5
cols = 4
edge_parameter = (1.0, 1.0 + 1.0j)
boundary_condition = (

onsite_parameter = 1.0
square_lattice = SquareLattice(
    rows=rows,
    cols=cols,
    edge_parameter=edge_parameter,
    onsite_parameter=onsite_parameter,
    boundary_condition=boundary_condition,

HyperCubicLattice¶

``HyperCubicLattice``은 ``LineLattice``와 ``SquareLattice``의 일반화된 형태이다. 이는 임의의 d차원 격자를 제공한다. 여기서, 우리는 3*4*5 크기의 3차원 격자를 예시로 구성해 본다. 격자의 사이즈는 튜플로 주어지며, 경계 조건들도 각 방향별로 명시할 수 있다. 이 예시에서, 모든 경계 조건은 열린 경계 조건이다.

[12]:

size = (3, 4, 5)
boundary_condition = (
    BoundaryCondition.OPEN,
    BoundaryCondition.OPEN,
    BoundaryCondition.OPEN,
)
cubic_lattice = HyperCubicLattice(size=size, boundary_condition=boundary_condition)

다음에서는 격자점들의 위치를 명시하는 3차원 입방체 격자를 그린다.

[13]:

# function for setting the positions
def indextocoord_3d(index: int, size: tuple, angle) -> list:
    z = index // (size[0] * size[1])
    a = index % (size[0] * size[1])
    y = a // size[0]
    x = a % size[0]
    vec_x = np.array([1, 0])
    vec_y = np.array([np.cos(angle), np.sin(angle)])
    vec_z = np.array([0, 1])
    return_coord = x * vec_x + y * vec_y + z * vec_z
    return return_coord.tolist()


pos = dict([(index, indextocoord_3d(index, size, angle=pi / 4)) for index in range(np.prod(size))])

cubic_lattice.draw(style=LatticeDrawStyle(pos=pos))

TriangularLattice¶

TriangularLattice``는 대각선 모서리들이 있는 2차원 격자로 볼 수 있는 삼각형 격자를 제공한다. ``boundary_condition 인자는 “open” 또는 “periodic”의 값을 가질 수 있다.

[14]:

rows = 4
cols = 3
boundary_condition = BoundaryCondition.OPEN

triangular_lattice = TriangularLattice(rows=rows, cols=cols, boundary_condition=boundary_condition)

triangular_lattice.draw()

[15]:

rows = 4
cols = 3
boundary_condition = BoundaryCondition.PERIODIC

triangular_lattice = TriangularLattice(rows=rows, cols=cols, boundary_condition=boundary_condition)

triangular_lattice.draw()

일반적인 격자¶

위에서, 우리는 평행 이동에 의해 불변인 격자들을 보았다. 여기서는, 일반적인 격자를 생각한다. 우리는 PyGraph의 인스턴스를 통해 꼭짓점들과 모서리들로 이루어진 일반적인 격자를 구성할 수 있다.

[16]:

graph = rx.PyGraph(multigraph=False)  # multiigraph shoud be False

    (4, 4, 3.0),
    (2, 5, -1.0),

# make a lattice

페르미-허바드 모형¶

페르미-허바드 (Fermi-Hubbard) 모델은 같은 위치에서만 상호작용이 존재하는 격자 상 전자들의 움직임을 나타나는 가장 간단한 모형이다. 해밀토니안은 다음과 같이 주어진다.

\[H = \sum_{i, j}\sum_{\sigma = \uparrow, \downarrow} t_{i, j} c_{i, \sigma}^\dagger c_{j, \sigma} + U \sum_{i} n_{i, \uparrow} n_{i, \downarrow},\]

여기서 \(c_{i, \sigma}^\dagger\) 와 \(c_{i, \sigma}\) 에 있는 스핀 \(\sigma\) 는 수 연산자로, \(n_{i, \sigma} = c_{i, \sigma}^\dagger c_{i, \sigma}\) 에 의해 정의된다. 행렬 \(t_{i, j}\) 는 상호작용 행렬이라 불리는 에르미트 행렬이다. 인자 \(U\) 는 상호작용의 크기를 나타낸다.

우리는 FermiHubbardModel 클래스를 활용해 주어진 격자에 대응되는 해밀토니안을 생성할 수 있다. 여기서, 우리는 2차원 격자에서 균일한 상호작용과 상호작용 인자들을 갖는 해밀토니안을 구성한다.

[20]:

square_lattice = SquareLattice(rows=5, cols=4, boundary_condition=BoundaryCondition.PERIODIC)

t = -1.0  # the interaction parameter
v = 0.0  # the onsite potential
u = 5.0  # the interaction parameter U

fhm = FermiHubbardModel.uniform_parameters(
    lattice=square_lattice,
    uniform_interaction=t,
    uniform_onsite_potential=v,
    onsite_interaction=u,
)

해밀토니안을 페르미온형 (fermionic) 연산자들로 나타내기 위해서, 우리는 second_q_ops 메소드를 사용한다. 해밀토니안은 FermionicOp 의 인스턴스로 반환된다.

  • 참고

    • The number of fermionic operators required is twice the number of lattice sites because of the spin degrees of freedom.

    • In the implementation, even indexes correspond to up-spin and odd indexes to down-spin.

[21]:

ham = fhm.second_q_ops(display_format="sparse").reduce()
print(ham)

Lattice 는 모서리들에 가중치를 갖고 있어, 우리는 Lattice 인스턴스를 활용해 일반적인 상호작용 행렬을 정의할 수 있다. 여기서, 우리는 균일하지 않은 상호작용 인자들이 주어진 일반적인 격자에서 페르미-허바드 모델을 고려한다. 이 경우, 격자 상의 가중치들은 상호작용 행렬로 생각할 수 있다. 해밀토니안을 생성한 후에는 (second_q_ops), 큐비트 변환기를 이용해 큐비트 연산자들을 생성하고 활용 가능한 적당한 알고리즘을 이용해 대응되는 격자 문제를 풀 수 있다.

[22]:

graph = rx.PyGraph(multigraph=False)  # multiigraph shoud be False
graph.add_nodes_from(range(6))
weighted_edge_list = [
    (0, 1, 1.0 + 1.0j),
    (0, 2, -1.0),
    (2, 3, 2.0),
    (4, 2, -1.0 + 2.0j),
    (4, 4, 3.0),
    (2, 5, -1.0),
]
graph.add_edges_from(weighted_edge_list)

general_lattice = Lattice(graph)  # the lattice whose weights are seen as the interaction matrix.
u = 5.0  # the interaction parameter U

fhm = FermiHubbardModel(lattice=general_lattice, onsite_interaction=u)

ham = fhm.second_q_ops(display_format="sparse").reduce()
ham.set_truncation(0)
print(ham)
