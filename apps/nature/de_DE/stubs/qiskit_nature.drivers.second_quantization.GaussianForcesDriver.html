GaussianForcesDriver¶

class GaussianForcesDriver(jcf='\n#p B3LYP/cc-pVTZ Freq=(Anharm) Int=Ultrafine SCF=VeryTight\n\nCO2 geometry optimization B3LYP/cc-pVTZ\n\n0 1\nC        -0.848629    2.067624    0.160992\nO         0.098816    2.655801   -0.159738\nO        -1.796073    1.479446    0.481721\n\n', logfile=None, normalize=True)[Quellcode]¶

Bases: VibrationalStructureDriver

Gaussian™ 16 forces driver.

Parameter

  • jcf – A job control file conforming to Gaussian™ 16 format. This can be provided as a single string with ‚\n‘ line separators or as a list of strings.

  • logfile – Instead of a job control file a log as output from running such a file can optionally be given.

  • normalize – Whether to normalize the factors used in creation of the VibrationalEnergy as returned when this driver is run.

Verursacht

QiskitNatureError – If jcf given and Gaussian™ 16 executable cannot be located.
