Getting started¶

Installation¶

Qiskit Optimization 是基於 Qiskit 主套件的，主套件有自己的 Qiskit 入門指南 ，其中詳細說明了 Qiskit 的安裝選項及其支援的環境/平臺。您應該首先參考 Qiskit 主套件入門指南。之後，按照這裡的訊息，額外完成 Qiskit Optimization 套件的安裝。

由於某些原始碼或程式尚未（或無法）在當前的預設情況下被安裝，Qiskit Optimization 裡有些功能被設定為選用安裝。這包括一些處理分子問題的古典程式庫與程式。請參閱：選用安裝 了解更多資訊。

The simplest way to get started is to follow the getting started 『Start locally』 for Qiskit here Qiskit Getting Started

在您安裝有 Qiskit 的虛擬環境中，類似於安裝 Qiskit visualization 支援的方式，您可以將 optimization 新增至額外列表中，例如：

pip install qiskit[optimization]

需要注意的是，如果您是 zsh 使用者（在新版本的 macOS 中是預設 Shell），則需要將 qiskit[optimization] 放置在引號中：

pip install 'qiskit[optimization]'

從來源安裝 Qiskit Optimization 可讓您存取開發中的最新版本，而不是使用 Python Package Index (PyPI) 儲存庫中的版本。這能讓您更有效率地檢查及擴展最新版 Qiskit Optimization 程式碼。

由於 Qiskit Optimization 依賴於 Qiskit，Qiskit Optimization 中最新的變動可能需要使用 Qiskit 中新增或更改的功能，所以您需要先遵循 `Qiskit 入門指南<https://qiskit.org/documentation/getting_started.html>`__ 中 「從來源安裝」 的指令。

Installing Qiskit Optimization from Source

使用已安裝有 Qiskit 的開發環境，即可安裝 Qiskit Optimization。

  1. 複製 Qiskit Optimization 儲存庫。

    git clone https://github.com/Qiskit/qiskit-optimization.git

  2. 複製儲存庫時，會建立名為 qiskit-optimization 的本地端資料夾。

    cd qiskit-optimization

  3. If you want to run tests or linting checks, install the developer requirements.

    pip install -r requirements-dev.txt

  4. 安裝 qiskit-optimization。

    pip install .

If you want to install it in editable mode, meaning that code changes to the project don’t require a reinstall to be applied, you can do this with:

pip install -e .

選用安裝¶

  • IBM CPLEX 可使用 ``pip install 『qiskit-optimization[cplex]』` ` 來安裝，以啟用 ` LP` 檔案的讀取，以及 ` cclexOptimizer ` 的用法， ` `cclex.Cplex `. 目前沒有 python 3.9 版本的 CPLEX。 在此情況下， CPLEX 安裝指令將沒有作用。

  • CVXPY 可以使用指令 pip install 'qiskit-optimization[cvx]' ` 。安裝的 CVXPY 將啟用 Goemans-Williamson 演算法，作為最佳化因子 `GoemansWilliamsonOptimizer `。

  • Matplotlib 可以使用指令 pip install 'qiskit-optimization[matplotlib]' 。 Matplotlib 的安裝將在圖形最佳化應用程式類別中啟用 `rain 的用法。

  • Gurobipy 可使用指令 pip install 'qiskit-optimization[gurobi]' `. Gurobipy 安裝後將啟用 `GurobiOptimizer』 的使用。
