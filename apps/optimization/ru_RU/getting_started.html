Начало работы¶

Инструкция по установке¶

Пакет Qiskit Optimization зависит от основного пакета Qiskit, который имеет свою собственную страницу Qiskit Getting Started <https://qiskit.org/documentation/getting_started.html> __ с подробным описанием опций установки Qiskit и поддерживаемых сред/платформ. Обратите на это внимание в первую очередь. Далее здесь можно найти информацию, которая сфокусируется на дополнительной установке, специфичной для Qiskit Optimization.

В Qiskit Optimization есть некоторые функции, которые были сделаны необязательными, если зависимые программы и/или программы поддержки не установлены (или не могут быть установлены) по умолчанию. Это IBM CPLEX, CVXPY и Matplotlib. Дополнительная информация приведена в разделе Необязательные для установки компоненты.

Самый простой способ - начать с раздела „Запуск локально“ для Qiskit здесь Qiskit Getting Started

В Вашей виртуальной среде, где Вы установили Qiskit, просто добавьте optimization в дополнительный список пакетов подобно тому как устанавливается дополнительная поддержка visualization в Qiskit, например:

pip install qiskit[optimization]

Стоит отметить, что если Вы используете zsh (оболочка по умолчанию для более новых версий macOS), Вам нужно указать qiskit[optimization] в кавычках:

pip install 'qiskit[optimization]'

Установка элементов из исходного кода позволит Вам получить доступ к последней обновленной версии Qiskit Optimization вместо использования версии из репозитория Python Package Index (PyPI). Это даст Вам возможность более эффективно изучить и внести свой вклад в улучшение кода последней версии Qiskit Optimization.

Так как Qiskit Optimization зависит от Qiskit, и его последние изменения могут потребовать новых или измененных функций Qiskit, сначала следуйте инструкциям «Установить из исходного кода» здесь Qiskit Getting Started

Installing Qiskit Optimization from Source

Используя ту же среду разработки, в которой Вы установили Qiskit, Вы можете установить пакет Qiskit Optimization.

  1. Клонируйте репозиторий Qiskit Optimization.

    git clone https://github.com/Qiskit/qiskit-optimization.git

  2. Клонирование репозитория создает локальную папку с названием qiskit-optimization.

    cd qiskit-optimization

  3. Если Вы хотите запустить тесты, установите необходимые инструменты для разработчика.

    pip install -r requirements-dev.txt

  4. Установите qiskit-optimization.

    pip install .

Если Вы хотите установить пакет в редактируемом режиме, который позволяет не переустанавливать пакет в случае изменения кода в проекте, Вы можете сделать это с помощью:

pip install -e .

Необязательные для установки компоненты¶

  • ** IBM CPLEX* * может быть установлен с помощью команды pip install 'qiskit-optimization[cplex]' для включения чтения LP файлов и использования CplexOptimizer, оболочки для ``cplex.Cplex ``. В настоящее время нет версии CPLEX под Python 3.9. В этом случае команда установки CPLEX не сработает.

  • CVXPY можно установить с помощью команды pip install 'qiskit-optimization[cvx]'. Установка CVXPY позволит использовать алгоритм Goemans-Williamson в качестве оптимизатора GoemansWilliamsonOptimizer.

  • Matplotlib можно установить с помощью команды pip install 'qiskit-optimization[matplotlib]'. Установка Matplotlib позволит использовать метод draw в приложениях по оптимизации графов.

  • Gurobipy может быть установлен с помощью команды pip install 'qiskit-optimization[gurobi]'. Gurobipy позволяет использовать GurobiOptimizer.
