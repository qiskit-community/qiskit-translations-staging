Premiers Pas¶

Installation¶

Qiskit Machine Learning dépend du package Qiskit principal qui a son propre  » Qiskit Getting Started <https://qiskit.org/documentation/getting_started.html>` __ détaillant les options d’installation de Qiskit et de ses environnements et plateformes pris en charge. Vous devriez d’abord vous référer à cela. Ensuite, les informations ci-dessous peuvent être suivies qui concernent l’installation supplémentaire spécifique à Qiskit Machine Learning.

Qiskit Machine Learning a certaines fonctions qui ont été rendues optionnelles lorsque le code dépendant et/ou le(s) programme(s) de support(s) ne sont pas (ou ne peuvent pas être) installé(s) par défaut. Il s’agit de PyTorch et Sparse. Pour plus d’informations, voir Installations optionnelles.

La façon la plus simple de commencer est de suivre l’initiation “Démarrer localement” pour Qiskit ici Qiskit Getting Started <https://qiskit.org/documentation/getting_started.html> __

Dans l’environnement virtuel où vous avez installé Qiskit, ajoutez simplement machine-learning à la liste supplémentaire de manière similaire à la façon dont le support de visualisation supplémentaire est installé pour Qiskit, c’est-à-dire:

pip install qiskit[machine-learning]

Il est utile de préciser que si vous êtes un utilisateur de zsh (qui est le shell par défaut dans les nouvelles versions de macOS) que vous devrez mettre qiskit[machine-learning] entre guillemets :

pip install 'qiskit[machine-learning]'

Installer des éléments depuis la source vous permet d’utiliser la version la plus à jour de Qiskit, plutôt que d’utiliser la version du dépôt de PyPI (Python Package Index). Ceci vous donnera la possibilité d’inspecter et d’étendre plus efficacement la dernière version du code Qiskit.

Puisque Qiskit Machine Learning dépend de Qiskit, et que ses dernières modifications peuvent nécessiter des fonctionnalités nouvelles ou modifiées de Qiskit, vous devez d’abord suivre l’installation de Qiskit « Installer à partir de la source » qui se trouve à cet endroit : Qiskit Getting Started <https://qiskit.org/documentation/getting_started.html> __

Installing Qiskit Machine Learning from Source

En utilisant l’environnement de développement dans lequel vous avez installé Qiskit, vous êtes prêt à installer Qiskit Machine Learning.

  1. Clonez le référentiel Qiskit Machine Learning.

    git clone https://github.com/Qiskit/qiskit-machine-learning.git

  2. Le clonage du référentiel crée un dossier local appelé qiskit-machine-learning.

    cd qiskit-machine-learning

  3. Si vous souhaitez réaliser des tests ou faire de l’analyse de code source (linting), installez les pré-requis Developeur.

    pip install -r requirements-dev.txt

  4. Installez qiskit-machine-learning.

    pip install .

Pour une installation en mode Editable (les changements dans le code sont alors pris en compte sans besoin de réinstaller le projet), vous pouvez choisir cette commande :

pip install -e .

Installations optionnelles¶

  • PyTorch, peut être installé soit en utilisant la commande pip install “qiskit-machine-learning[torch]” pour installer le package, soit en se référant à PyTorch `getting started<https://pytorch.org/get-started/locally/>`__. Lorsque PyTorch est installé, le TorchConnector facilite l’utilisation des réseaux quantiques.

  • ** Sparse**, peut être installé à l’aide de la commande pip install’qiskit-machine-learning[sparse]”. L’installation de Sparse va permettre l’utilisation de tableaux et /ou tenseurs épars.
