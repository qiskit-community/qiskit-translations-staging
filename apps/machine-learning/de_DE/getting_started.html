Installation¶

Qiskit Machine Learning hängt vom Qiskit-Hauptpaket ab, das einen eigenen Abschnitt ` Qiskit-Erste Schritte <https://qiskit.org/documentation/getting_started.html>` __ hat, der die Installationsoptionen für Qiskit und die unterstützten Umgebungen/Plattformen beschreibt. Dieses sollten Sie zuerst lesen. Dann können die Informationen hier nachgelesen werden, die sich auf die zusätzliche Installation speziell für Qiskit Machine Learning beziehen.

Qiskit Machine Learning hat einige Funktionen, die optional gemacht wurden, für den Fall dass der abhängige Code und/oder die Unterstützungsprogramme nicht standardmäßig installiert sind (oder nicht installiert werden können). Dabei handelt es sich um PyTorch und Sparse. Siehe auch Optionale Installationen für weitere Informationen.

Die einfachste Art loszulegen ist, der Anleitung ‚Lokal starten‘ für Qiskit hier zu folgen Qiskit Erste Schritte

In Ihrer virtuellen Umgebung, in der Sie Qiskit installiert haben, fügen Sie einfach ` ` machine-learning ` ` zur Extra-Liste hinzu, ähnlich wie die extra Unterstützung von ` ` Visualisierung ` ` für Qiskit installiert wird, also:

pip install qiskit[machine-learning]

Als ein Benutzer von zsh (was die Standard-Shell der neueren Versionen von macOS ist) muss man qiskit[machine-learning] in Anführungszeichen setzen:

pip install 'qiskit[machine-learning]'

Installing Qiskit Machine Learning from source allows you to access the most recently updated version under development instead of using the version in the Python Package Index (PyPI) repository. This will give you the ability to inspect and extend the latest version of the Qiskit Machine Learning code more efficiently.

Da Qiskit Machine Learning von Qiskit abhängt und due neuesten Änderungen möglicherweise neue oder geänderte Qiskit-Funktionen erfordern, folgen Sie zuerst den Anweisungen zu Qiskit „Installieren aus dem Quelltext“ hier Qiskit Erste Schritte

Installing Qiskit Machine Learning from Source

Mit derselben Entwicklungsumgebung, in der Sie Qiskit installiert haben, können Sie Qiskit Machine Learning installieren.

  1. Klonen Sie das Qiskit-Repository für Machine Learning.

    git clone https://github.com/Qiskit/qiskit-machine-learning.git

  2. Klonen des Repository erzeugt einen lokalen Ordner namens ` ` qiskit-machine-learning ` `.

    cd qiskit-machine-learning

  3. Wenn Sie Tests oder Linting-Prüfungen ausführen möchten, installieren Sie die Entwicklungsabhängigkeiten.

    pip install -r requirements-dev.txt

  4. Installation von ` ` qiskit-machine-learning ` `.

    pip install .

Wenn Sie im editierbaren Modus installieren möchten, in dem Änderungen des Codes im Projekt keine Neuinstallation erfordern, können Sie das folgendermaßen tun:

pip install -e .

Optionale Installationen¶

  • PyTorch kann installiert werden, indem man entweder der PyTorch get started Anleitung folgt, oder die benötigten Pakete mit dem Befehl pip install ‚qiskit-machine-learning[torch]‘ einbindet bzw. auf sie verweist. Wenn PyTorch verfügbar ist, ermöglicht der TorchConnector die Verwendung von Quanten auf Netzwerken.

  • ** Sparse**, kann mit dem Befehl ` pip install ‚qiskit-machine-learning[sparse]‘ ` installiert werden. Die Installation von Sparse ermöglicht die Verwendung von schwach besetzten Arrays bzw. Tensoren.
