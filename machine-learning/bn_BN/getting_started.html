শুরুর নির্দেশনা¶

স্থাপন (ইনস্টলেশন)¶

Qiskit মেশিন লার্নিং মূল Qiskit প্যাকেজের উপর নির্ভর করে যার নিজস্ব `Qiskit শুরু করা <https://qiskit.org/docamentation/getting_start.html> `__ Qiskit এবং এর সমর্থিত পরিবেশ / প্ল্যাটফর্মগুলির জন্য ইনস্টলেশন বিকল্পগুলি বিশদ বর্ণনা করে। আপনার প্রথমে উল্লেখ করা উচিত। তারপরে এখানকার তথ্য অনুসরণ করা যেতে পারে যা কিসকিট মেশিন লার্নিং এর সাথে সম্পর্কিত অতিরিক্ত ইনস্টলেশনকে কেন্দ্র করে।

Qiskit মেশিন লার্নিংয়ের কিছু ফাংশন রয়েছে যা ঐচ্ছিকভাবে তৈরি করা হয়েছে যেখানে নির্ভরযোগ্য কোড এবং / বা সমর্থন প্রোগ্রাম (গুলি) ডিফল্টরূপে ইনস্টল করা হয় না (বা করা যায় না)। এগুলি হল পাইটর্চ (PyTorch) এবং স্পারস (Sparse)। আরও তথ্যের জন্য দেখুন: রেফ: optional_installs।

সহজ করে শুরু করার উপায় হ'ল 'Start locally' থেকে শুরু করুন , Qiskit Getting Started

আপনার ভার্চুয়াল পরিবেশে যেখানে আপনি কিস্কিট (Qiskit) ইনস্টল করেছেন, কিস্কিটের জন্য কীভাবে অতিরিক্ত visualization সমর্থনটি ইনস্টল করা হয়েছে তার সাথে একইভাবে অতিরিক্ত তালিকায় machine-learning যুক্ত করুন:

pip install qiskit[machine-learning]

মনে করিয়ে দেয়া প্রয়োজন, আপনি যদি zsh (ম্যাকওএস এর নতুনতর সংস্করণগুলোতে এটি পূর্বনির্ধারিত (ডিফল্ট) শেল) ব্যবহারকারী হয়ে থাকেন তাহলে আপনাকে উদ্ধৃতির মধ্যে qiskit[machine-learning] লিখতে হবে:

pip install 'qiskit[machine-learning]'

উৎস থেকে উপাদানগুলি ইনস্টল করলে আপনি Qiskit এর সর্বাধিক আপডেট হওয়া সংস্করণটি ব্যবহার করতে পারবেন Python Package Index (PyPI) রিপোসিটোরি থেকে ইনস্টল করার পরিবর্তে। এটি আপনাকে Qiskit মেশিন লার্নিং কোডের সর্বশেষ সংস্করণটি আরও নিখুঁত ভাবে পরিদর্শন করার এবং প্রসারিত করার দক্ষতা দেবে।.

যেহেতু কিস্কিট মেশিন লার্নিং কিস্কিটের উপর নির্ভর করে এবং এর সর্বশেষ পরিবর্তনগুলিতে কিস্কিটের নতুন বা পরিবর্তিত বৈশিষ্ট্যগুলির প্রয়োজন হতে পারে, আপনার প্রথমে কিস্কিটের "Install from source" `Qiskit Getting Started <https://qiskit.org/documentation/getting_started.html> নির্দেশাবলী এখানে অনুসরণ করা উচিত।

Installing Qiskit Machine Learning from Source

আপনি যেই কিস্কিট ইনস্টল করেছেন সেই একই পরিবেশটি ব্যবহার করে আপনি কিস্কিট মেশিন লার্নিং ইনস্টল করতে প্রস্তুত।

  1. Qiskit মেশিন লার্নিং রিপোজিটরি এর প্রতিলিপি (Clone) করুন

    git clone https://github.com/Qiskit/qiskit-machine-learning.git

  2. রিপোসিটোরিটিকে ক্লোন করলে qiskit-machine-learning নামক একটি স্থানীয় ফোল্ডার তৈরী হবে।

    cd qiskit-machine-learning

  3. টেস্ট রান করানোর জন্য বা মোলায়েম করা (লিনটিং) চেক করার জন্য, ডেভেলপারের প্রয়োজন ইনস্টল করুন।

    pip install -r requirements-dev.txt

  4. qiskit-machine-learning ইনস্টল করুন।

    pip install .

আপনি যদি এটিকে সম্পাদনযোগ্য মোডে ইনস্টল করতে চান, মানে এই প্রজেক্টের কোড পরিবর্তনের জন্য প্রয়োগ করার জন্য পুনরায় ইনস্টল করা প্রয়োজন নেই, আপনি এটি করতে পারেন:

pip install -e .

ঐচ্ছিক ইনস্টল¶

  • ** PyTorch **, প্যাকেজ ইনস্টল করার জন্য pip install 'qiskit-machine-learning[torch]' কমান্ড ব্যবহার করে ইনস্টল করা যেতে পারে অথবা PyTorch getting started পড়ুন। যখন PyTorch ইনস্টল করা হয়, TorchConnector কোয়ান্টাম কম্পিউটেড নেটওয়ার্ক ব্যবহার করতে সুবিধা করে।

  • pip install 'qiskit-machine-learning[sparse]' কমান্ড ব্যবহার করে Sparse স্থাপন করা যেতে পারে| Sparse স্থাপন করলে sparse arrays/tensors ব্যবহার করা সম্ভব হবে|
