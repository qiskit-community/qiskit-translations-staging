시작하기¶

설치¶

Qiskit Machine Learning은 Qiskit및 지원되는 환경/플랫폼의 설치 옵션을 자세히 설명하는 Qiskit Getting Started 를 가지고 있는 기본 Qiskit 패키지에 의존한다. 먼저 그것을 참고해야 한다. 그 후 Qiskit Machine Learning의 추가 설치에 초점을 맞추는 이 내용을 따라라.

Qiskit Machine Learning에는 종속 코드 및/또는 지원 프로그램이 기본적으로 설치되어 있지 않은 (또는 설치될 수 없는) 일부 선택적인 기능들이 있다. 이에는 PyTorch와 Sparse가 있다. 자세한 내용은 선택적 설치 를 참조하여라.

시작할 수 있는 가장 간단한 방법은 Qiskit 에서 〈로컬로 시작하기〉 `Qiskit 시작하기<https://qiskit.org/documentation/getting_started.html>`__를 따르는 것이다.

Qiskit을 설치한 가상 환경에서 추가적인 visualization 지원을 설치한 방법과 마찬가지로 machine-learning 을 단순히 extra 리스트에 추가한다.

pip install qiskit[machine-learning]

Zsh 사용자 (최신 버전의 macOS 기본 쉘) 인 경우 qiskit[machine-learning] 을 큰 따옴표로 묶어야 한다:

pip install 'qiskit[machine-learning]'

파이썬 패키지 인덱스 (PyPI) 저장소를 이용하여 설치하는 대신 소스 코드로부터 Qiskit 머신러닝을 설치하면 개발중인 가장 최근 업데이트된 버전을 사용할 수 있다. 이를 통해 손쉽게 Qiskit 머신러닝의 최신 코드를 들여다 보고 확장할 수 있다.

Qiskit Machine Learning은 Qiskit에 의존하므로 최신 변경 사항에는 Qiskit의 새로운 기능이나 변경된 기능이 필요할 수 있다. 따라서 다음 문서에 있는 Qiskit의 《소스 코드로 설치》 의 지시사항을 먼저 수행해야 한다: Qiskit 시작하기

Installing Qiskit Machine Learning from Source

Qiskit을 설치한 동일한 개발 환경을 사용하여 Qiskit Machine Learning을 설치할 준비가 되었다.

  1. Qiskit Machine Learning 저장소를 복제한다.

    git clone https://github.com/Qiskit/qiskit-machine-learning.git

  2. 저장소를 복제하면 ``qiskit-machine-learning``라는 이름의 로컬 디렉토리가 생성된다.

    cd qiskit-machine-learning

  3. 만약 테스트 코드를 실행하거나 철자법 검사를 하고자 한다면, 개발자 요구사항 라이브러리를 설치하라.

    pip install -r requirements-dev.txt

  4. ``qiskit-machine-learning``을 설치한다.

    pip install .

만약, 프로젝트에 코드가 변화하였을 때 재설치가 필요하지 않는, 수정 가능한 모드로 설치하고자 한다면, 다음의 명령어를 사용하라.

pip install -e .

선택적 설치¶

  • PyTorch 는 pip install 〈qiskit-machine-learning[torch]〉 명령어를 사용하여 패키지를 설치하거나 PyTorch의 시작하기 를 참조하여 설치할 수 있다. PyTorch가 설치되면, TorchConnector 가 양자 컴퓨팅 네트워크를 사용할 수 있도록 해 준다.

  • Sparse 는 pip install 〈qiskit-machine-learning[sparse]〉 명령어를 사용하여 설치할 수 있다. Sparse를 설치하면 희소 배열/텐서를 사용할 수 있다.
