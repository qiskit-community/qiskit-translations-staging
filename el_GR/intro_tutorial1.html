Εισαγωγή στο Qiskit¶

Όταν χρησιμοποιείτε το Qiskit, μια ροή εργασιών χρήστη αποτελείται από τα ακόλουθα τέσσερα βήματα:

  • build: Σχεδιάστε ένα κβαντικό κύκλωμα (ή κυκλώματα) που αντιπροσωπεύει το πρόβλημα που εξετάζετε.

  • Compile: Μεταγλωττίσετε κυκλώματα για μια συγκεκριμένη κβαντική υπηρεσία, π.χ. ένα κβαντικό σύστημα ή κλασικό προσομοιωτή.

  • Run: Εκτελέστε τα μεταγλωττισμένα κυκλώματα στις καθορισμένες κβαντικές υπηρεσία(ες). Αυτές οι υπηρεσίες μπορούν να βασίζονται σε cloud ή τοπικά.

  • Analyze: Υπολογισμός στατιστικών στιχείων και απεικόνιση των αποτελεσμάτων των πειραμάτων.

Εδώ είναι ένα παράδειγμα ολόκληρης της ροής εργασίας, με κάθε βήμα να εξηγείται λεπτομερώς στις επόμενες ενότητες:

import numpy as np
from qiskit import QuantumCircuit, transpile
from qiskit.providers.aer import QasmSimulator
from qiskit.visualization import plot_histogram

# Use Aer's qasm_simulator
simulator = QasmSimulator()

# Create a Quantum Circuit acting on the q register
circuit = QuantumCircuit(2, 2)

# Add a H gate on qubit 0
circuit.h(0)

# Add a CX (CNOT) gate on control qubit 0 and target qubit 1
circuit.cx(0, 1)

# Map the quantum measurement to the classical bits
circuit.measure([0,1], [0,1])

# compile the circuit down to low-level QASM instructions
# supported by the backend (not needed for simple circuits)
compiled_circuit = transpile(circuit, simulator)

# Execute the circuit on the qasm simulator
job = simulator.run(compiled_circuit, shots=1000)

# Grab results from the job
result = job.result()

# Returns counts
counts = result.get_counts(compiled_circuit)
print("\nTotal count for 00 and 11 are:",counts)

# Draw the circuit
circuit.draw()


Total count for 00 and 11 are: {'11': 483, '00': 517}

     └───┘┌─┴─┐└╥┘┌─┐
q_1: ─────┤ X ├─╫─┤M├

                0  1 
# Plot a histogram
plot_histogram(counts)

Ροής Εργασίας Βήμα-Βήμα¶

Το παραπάνω πρόγραμμα μπορεί να χωριστεί σε έξι βήματα:

  1. Εισαγωγή πακέτων

  2. Αρχικοποίηση μεταβλητών

  3. Προσθήκη πυλών

  4. Απεικόνιση του κυκλώματος

  5. Προσομοίωση του πειράματος

  6. Απεικόνιση των αποτελεσμάτων

Βήμα 1: Εισαγωγή Πακέτων¶

Τα βασικά στοιχεία που απαιτούνται για το πρόγραμμά σας εισάγονται ως εξής:

import numpy as np
from qiskit import QuantumCircuit
from qiskit.providers.aer import QasmSimulator
from qiskit.visualization import plot_histogram

Πιο αναλυτικά, οι εισαγωγές είναι

  • QuantumCircuit: μπορεί να θεωρηθεί ως οδηγίες του κβαντικού συστήματος. Κρατάει όλες τις κβαντικές σου λειτουργίες.

  • QasmSimulator: είναι ο προσομοιωτής κυκλώματος υψηλής απόδοσης Aer.

  • plot_histogram: δημιουργεί ιστογράμματα.

Βήμα 2: Αρχικοποίηση Μεταβλητών¶

Εξετάστε την επόμενη γραμμή κώδικα

circuit = QuantumCircuit(2, 2)

Εδώ, αρχικοποιείτε με 2 qubits σε μηδενική κατάσταση; με 2 κλασικά bits με μηδέν; και circuit είναι το κβαντικό κύκλωμα.

Σύνταξη:

  • QuantumCircuit(int, int)

Βήμα 3: Προσθήκη Πυλών¶

Μπορείτε να προσθέσετε πύλες (λειτουργίες) για να χειριστείτε τους καταχωρητές του κυκλώματός σας.

Εξετάστε τις ακόλουθες τρεις γραμμές κώδικα:

circuit.h(0)
circuit.cx(0, 1)
circuit.measure([0,1], [0,1])

Οι πύλες προστίθενται στο κύκλωμα μία προς μία για να σχηματίσουν την κατάσταση Bell

\[|\psi\rangle = \left(|00\rangle+|11\rangle\right)/\sqrt{2}.\]

Ο παραπάνω κώδικας εφαρμόζει τις ακόλουθες πύλες:

  • QuantumCircuit.h(0): Μια πύλη Hadamard \(H\) στο qubit 0, η οποία την βάζει σε κατάσταση superposition.

  • QuantumCircuit.cx(0, 1): Μια controlled-Not λειτουργία (\(CNOT\)) στο qubit ελέγχου 0 και στο qubit στόχου 1, τοποθετώντας τα qubits σε κατάσταση entangled.

  • QuantumCircuit. easure([0,1], [0,1]): αν περάσεις όλα τα κβαντικά και κλασικά registers στο measure, το αποτέλεσμα μέτρησης του x qubit θα αποθηκευτεί στο αντίστοιχο κλασικό x bit.

Βήμα 4: Απεικόνιση του κυκλώματος¶

Μπορείς να χρησιμοποιήσεις το qiskit.circuit.QuantumCircuit.draw() για να δεις το κύκλωμα που έχεις σχεδιάσει με τις διάφορες μορφές που χρησιμοποιούνται σε πολλά εγχειρίδια και ερευνητικά άρθρα.

circuit.draw()

     ┌───┐     ┌─┐   
q_0: ┤ H ├──■──┤M├───
     └───┘┌─┴─┐└╥┘┌─┐
q_1: ─────┤ X ├─╫─┤M├
          └───┘ ║ └╥┘
c: 2/═══════════╩══╩═
                0  1

Σε αυτό το κύκλωμα, τα qubits ταξινομούνται με το qubit μηδέν στην κορυφή και το qubit ένα στο κάτω μέρος. Το κύκλωμα διαβάζεται αριστερά προς τα δεξιά, πράγμα που σημαίνει ότι οι πύλες που εφαρμόζονται νωρίτερα στο κύκλωμα εμφανίζονται πιο αριστερά.

Το προεπιλεγμένο σύστημα υποστήριξης για το QuantumCircuit.draw() ή qiskit.visualization.circuit_drawer() είναι το σύστημα υποστήριξης κειμένου. Ωστόσο, ανάλογα με το τοπικό περιβάλλον σας, μπορεί να θέλετε να αλλάξετε αυτές τις προεπιλογές σε κάτι καταλληλότερο για την περίπτωση χρήσης σας. Αυτό γίνεται με το αρχείο ρύθμισης χρήστη. Από προεπιλογή, το αρχείο ρυθμίσεων χρήστη θα πρέπει να βρίσκεται στο αρχείο ~/.qiskit/settings.conf και είναι ένα αρχείο .ini.

Για παράδειγμα, ένα αρχείο settings.conf για τη ρύθμιση ενός Matplotlib drawer είναι:

[default]
circuit_drawer = mpl

Μπορείτε να χρησιμοποιήσετε οποιοδήποτε από τα έγκυρα συστήματα υποστήριξης σχεδιασμού κυκλωμάτων ως την τιμή για αυτή τη ρύθμιση, αυτό περιλαμβάνει κείμενο, mpl, latex και latex_source.

Βήμα 5: Προσομοίωση του πειράματος¶

Qiskit Aer είναι ένα υψηλής απόδοσης προσομοιωτής πλαίσιο για κβαντικά κυκλώματα. Παρέχει αρκετά backends <apidoc/providers/aer/aer_backends.html> __ για την επίτευξη διαφορετικών στόχων προσομοίωσης.

Αν έχετε προβλήματα με την εγκατάσταση του Aer, μπορείτε εναλλακτικά να χρησιμοποιήσετε τον πάροχο Basic Aer αντικαθιστώντας το Aer με το BasicAer. Το βασικό Aer περιλαμβάνεται στο Qiskit Terra.

import numpy as np
from qiskit import QuantumCircuit, transpile
from qiskit.providers.basicaer import QasmSimulatorPy
...

Για να προσομοιώσετε αυτό το κύκλωμα, θα χρησιμοποιήσετε το qasm_simulator. Κάθε τρέξιμο αυτού του κυκλώματος θα παράξει είτε το bit string 00 ή 11.

simulator = QasmSimulator()
compiled_circuit = transpile(circuit, simulator)
job = simulator.run(compiled_circuit, shots=1000)
result = job.result()
counts = result.get_counts(circuit)
print("\nTotal count for 00 and 11 are:",counts)


Total count for 00 and 11 are: {'00': 483, '11': 517}

Όπως αναμένεται, η συμβολοσειρά bit εξόδου είναι 00 περίπου 50 τοις εκατό του χρόνου. Ο αριθμός των φορών που εκτελείται το κύκλωμα μπορεί να καθοριστεί μέσω του ορίσματος shots της μεθόδου execute. Ο αριθμός εκτελέσεων της προσομοίωσης ορίστηκε να είναι 1000 (η προεπιλογή είναι 1024).

Μόλις έχεις ένα αντικείμενο result, μπορείς να αποκτήσεις πρόσβαση στις μετρήσεις μέσω της μεθόδου get_counts(circuit). Αυτό σας δίνει τα συνολικά αποτελέσματα του πειράματος που τρέξατε.

Βήμα 6 : Απεικόνιση των αποτελεσμάτων¶

Το Qiskit παρέχει πολλές απεικονίσεις,

συμπεριλαμβανομένης της συνάρτησης plot_histogram, για να δείτε τα αποτελέσματά σας.

plot_histogram(counts)

Οι παρατηρούμενες πιθανότητες \(Pr(00)\) και \(Pr(11)\) υπολογίζονται με τη λήψη των αντίστοιχων μετρήσεων και τη διαίρεση με το συνολικό αριθμό των εκτελέσεων.

Σημείωση

Try changing the shots keyword in the run() method to see how the estimated probabilities change.
