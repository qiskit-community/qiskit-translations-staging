পোর্টফোলিও অপটিমাইজেশন¶

ভূমিকা¶

এই টিউটোরিয়াল দেখায় কীভাবে \(n\) সম্পদের জন্য নীচের মীন-ভ্যারিয়্যান্স পোর্টফোলিও অপ্টিমাইজেশান সমস্যাটি সমাধান করা যায়:

\[\begin{split}\begin{aligned} \min_{x \in \{0, 1\}^n} q x^T \Sigma x - \mu^T x\\ \text{subject to: } 1^T x = B \end{aligned}\end{split}\]

যেখানে আমরা নিম্নলিখিত স্বরলিপি ব্যবহার করি:

  • \(x \in \{0, 1\}^n\) বাইনারি সিদ্ধান্ত ভেরিয়েবলের ভেক্টরকে বোঝায়, যা কোন সম্পদ বাছাই করে তা নির্দেশ করে (\(x[i] = 1\)) এবং কোনটি বেছে নেবে না (\(x[i] = 0\)),

  • \(\mu \in \mathbb{R}^n\) সম্পদের প্রত্যাশিত রিটার্ন সংজ্ঞায়িত করে,

  • \(\Sigma \in \mathbb{R}^{n \times n}\) সম্পদের মধ্যে সমবায়িকাগুলি নির্দিষ্ট করে,

  • \(q > 0\) সিদ্ধান্ত গ্রহণকারীর ঝুঁকি ক্ষুধা নিয়ন্ত্রণ করে,

  • এবং :math: B দ্বারা বোঝায় বাজেটকে, অর্থাৎ :math: n থেকে নির্বাচিত হওয়া সম্পদের সংখ্যাকে।

আমরা নিম্নোক্ত সরলীকরণগুলি ধরে নিই: - সমস্ত সম্পদের একই দাম রয়েছে (1 এ নরমালাইযকৃত ), - সম্পূর্ণ বাজেট :math: B যা ব্যয় করতে হবে, অর্থাত্ একজন ব্যক্তিকে অবিকল :math: ` B` সম্পদ নির্বাচন করতে হবে।

সাম্যতার সীমাবদ্ধতা :math: 1^T x = B একটি পেনাল্টি শর্ত :math:` (1^T x - B)^2`-তে ম্যাপ করা হয় যা একটি পরামিতি দ্বারা স্কেল করা হয় এবং নৈর্ব্যক্তিক অন্বয় (অব্জেক্টিভ ফাংশন) থেকে বিয়োগ করা হয়। ফলস্বরূপ প্রাপ্ত সমস্যাটি হ্যামিলটোনিয়ানে ম্যাপ করা যেতে পারে যার ভিত্তি/ভূমি মান বা অবস্থা সর্বোত্তম সমাধানের সাথে মিলে যায়। এই নোটবুকটি দেখায় যে কীভাবে ভ্যারিয়েশনাল কোয়ান্টাম আইজেনসলভার (ভিকিউই) বা কোয়ান্টাম অ্যাপ্রক্সিমেইট অপ্টিমাইজেশন অ্যালগরিদম (কিউএওএ) ব্যবহার করে কোনও নির্দিষ্ট পরামিতি সেটের জন্য অনুকূল সমাধান সন্ধান করা যায় ।

এই সমস্যার জন্য বাস্তব কোয়ান্টাম হার্ডওয়্যারের পরীক্ষাগুলি নিম্নলিখিত গবেষণাপত্রে আলোচিত হয়েছে: Improving Variational Quantum Optimization using CVaR. Barkoutsos et al. 2019.

আমাদের হ্যামিলটনীয়ানের জন্য এখানে একটি অপারেটর উদাহরণ তৈরি করা হয়েছে। এক্ষেত্রে পলিসগুলি পোর্টফোলিও সমস্যা থেকে অনূদিত আইসিং হ্যামিলটোনিয়ান থেকে এসেছে। এই নোটবুকের জন্য আমরা একটি এলোমেলো পোর্টফোলিও সমস্যা ব্যবহার করি। বাস্তব অর্থনৈতিক ডাটা ব্যবহার করে একে বিস্তৃত করা বেশ সোজা-সাপ্টা, যেমনটা দেখানো হয়েছে এখানটায়: Loading and Processing Stock-Market Time-Series Data

                 end=datetime.datetime(2016,1,30),
                 seed=seed)

sigma = data.get_period_return_covariance_matrix()

def index_to_selection(i, num_assets):
    s = "{0:b}".format(i).rjust(num_assets)
    x = np.array([1 if s[i]=='1' else 0 for i in reversed(range(num_assets))])
    return x

def print_result(result):
    selection = result.x
    value = result.fval
    print('Optimal: selection {}, value {:.4f}'.format(selection, value))

    eigenstate = result.min_eigen_solver_result.eigenstate
    eigenvector = eigenstate if isinstance(eigenstate, np.ndarray) else eigenstate.to_matrix()
    probabilities = np.abs(eigenvector)**2
    i_sorted = reversed(np.argsort(probabilities))
    print('\n----------------- Full result ---------------------')
    print('selection\tvalue\t\tprobability')
    print('---------------------------------------------------')
    for i in i_sorted:
        x = index_to_selection(i, num_assets)
        value = QuadraticProgramToQubo().convert(qp).objective.evaluate(x)
        #value = portfolio.to_quadratic_program().objective.evaluate(x)
        probability = probabilities[i]
        print('%10s\t%.4f\t\t%.4f' %(x, value, probability))

NumPyMinimumEigensolver (একটি ধ্রুপদী তথ্যসূত্র (রেফারেন্স) হিসাবে)¶

সমস্যাটির সমাধান করা যাক। প্রথমে ধ্রুপদী পদ্ধতিতে…

কীভাবে গঠন বা নির্মাণ (বিল্ড) হয়েছিল তার নির্দিষ্টতা বিবেচনা না করেই আমরা এখন উপরে নির্মিত অপারেটরটি ব্যবহার করতে পারি। আমরা NumPyMinimumEigensolver এর জন্য অ্যালগরিদম সেট করেছি যাতে আমাদের একটি শাস্ত্রীয় তথ্যসূত্র (রেফারেন্স) থাকতে পারে। সমস্যাটি 'ইজিং' এর জন্য সেট করা আছে।কোয়ান্টাম কম্পিউটেশনের পরিবর্তে, ধ্রুপদিভাবে হিসেব করা হয় বলে এখানে ব্যাকএন্ডের প্রয়োজন হয় না। ফলাফলটি ডিকশনারি হিসাবে ফিরে আসে

ভি কিউ ই ব্যবহার করে সমাধান¶

সমস্যাটি সমাধানের জন্য আমরা এখন ভেরিয়েশনাল কোয়ান্টাম আইজেনসলভার (ভিকিউই) ব্যবহার করতে পারি। ব্যবহারের জন্য আমরা অপটিমাইজার (অনুকূলায়ক) এবং পরিবর্তনশীল (ভ্যারিয়েশনাল) ফর্মটি নির্দিষ্ট করব।

টীকা: আপনি ব্যাকএন্ডের নাম সরবরাহ করে বিভিন্ন ব্যাকএন্ডে পরিবর্তন করতে পারেন।

from qiskit.utils import algorithm_globals

algorithm_globals.random_seed = 1234

cobyla = COBYLA()

ry = TwoLocal(num_assets, 'ry', 'cz', reps=3, entanglement='full')
quantum_instance = QuantumInstance(backend=backend, seed_simulator=seed, seed_transpiler=seed)

কিউএওএ (QAOA) ব্যবহার করে সমাধান¶

আমরা এখানে কোয়ান্টাম অ্যাপ্রক্সিমেইট অপ্টিমাইজেশন অ্যালগরিদম কিউএওএ (QAOA) ব্যবহার করে পাওয়া একটি ফলাফলও দেখাই। এটি আরেকটি পরিবর্তনশীল (ভ্যারিয়েশনাল) অ্যালগরিদম এবং এটি ব্যবহার করে সমস্যার ভিত্তিতে তৈরি একটি অভ্যন্তরীণ পরিবর্তনশীল (ভ্যারিয়েশনাল) ফর্ম।
