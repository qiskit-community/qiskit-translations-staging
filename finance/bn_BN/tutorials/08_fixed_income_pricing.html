Pricing Fixed-Income Assets (নির্দিষ্ট আয় সম্পদ মূল্য নির্ধারণ)¶

ভূমিকা¶

প্রাসঙ্গিক সুদের হারগুলি বর্ণনা করে এমন বিতরণগুলি জেনে আমরা একটি নির্দিষ্ট-আয়ের সম্পদের মূল্য চাই। সম্পদের নগদ \(c_t\) প্রবাহ এবং যে তারিখে তারা ঘটে সেগুলি জানা যায়। সম্পত্তির মোট মূল্য \(V\) হল এর গড় মান:

\[V = \sum_{t=1}^T \frac{c_t}{(1+r_t)^t}\]

প্রতিটি নগদ প্রবাহকে শূন্য কুপন বন্ড হিসেবে বিবেচনা করা হয় যার সাথে সংশ্লিষ্ট সুদের হার \(r_t\) যা তার পরিপক্কতার উপর নির্ভর করে। ব্যবহারকারীকে অবশ্যই প্রতিটি \(r_t\) (সম্ভবত পারস্পরিক সম্পর্কযুক্ত) অনিশ্চয়তার বণ্টন মডেলিং নির্দিষ্ট করতে হবে এবং সেইসাথে প্রতিটি বন্টনের নমুনার জন্য তিনি যে কিউবিটগুলি ব্যবহার করতে চান তা উল্লেখ করতে হবে। এই উদাহরণে আমরা সম্পদের মান সুদের হারে প্রথম অর্ডারে প্রসারিত করি \(r_t\)। এটি সম্পদের সময়কাল অনুসারে অধ্যয়নের সাথে মিলে যায়। উদ্দেশ্য ফাংশনের আনুমানিকতা নিম্নলিখিত গবেষণা অনুসরণ করে: Quantum Risk Analysis. Woerner, Egger. 2018.

অনিশ্চয়তা মডেল¶

আমরা একটি বহুবিধ নর্মাল দৈব বণ্টনকে (multivariate normal random distribution) কোয়ান্টামের স্টেটে \(d\) মাত্রাগুলিতে লোড করার জন্য একটি বর্তনী (সার্কিট) ফ্যাক্টরি তৈরি করি । বিতরণটি একটি নির্দিষ্ট বিরতিতে \(\otimes_{i=1}^d [low_i, high_i]\) কাটা হয় এবং সেগুলি \(2^{n_i}\) গ্রিড পয়েন্ট ব্যবহার করে বিযুক্ত করা হয় যেখানে \(n_i\) দ্বারা \(i = 1,\ldots, d\) মাত্রার কিউবিট সংখ্যার বর্ণনা করা হচ্ছে। সার্কিট ফ্যাক্টরির অনুরূপ ইউনিটারি অপারেটর নিম্নলিখিতগুলিকে প্রয়োগ করে:

\[\big|0\rangle_{n_1}\ldots\big|0\rangle_{n_d} \mapsto \big|\psi\rangle = \sum_{i_1=0}^{2^n_-1}\ldots\sum_{i_d=0}^{2^n_-1} \sqrt{p_{i_1,...,i_d}}\big|i_1\rangle_{n_1}\ldots\big|i_d\rangle_{n_d},\]

যেখানে \(p_{i_1, ..., i_d}\) সংক্ষিপ্ত এবং বিযুক্ত বিতরণগুলির সম্ভাবনা বোঝায় এবং \(i_j\) অ্যাফাইন ম্যাপ ব্যবহার করে \([low_j, high_j]\) ডান বিরতি ব্যবধানে ম্যাপ করা হয়েছে:

\[\{0, \ldots, 2^{n_{j}}-1\} \ni i_j \mapsto \frac{high_j - low_j}{2^{n_j} - 1} * i_j + low_j \in [low_j, high_j].\]

অনিশ্চয়তা মডেল ছাড়াও, আমরা একটি অ্যাফাইন মানচিত্রও প্রয়োগ করতে পারি, উদা: একটি মূল উপাদান বিশ্লেষণের (principal component analysis) ফল। ব্যবহৃত সুদের হারগুলি এভাবে দেওয়া হয়:

\[\vec{r} = A * \vec{x} + b,\]

যেখানে \(\vec{x} \in \otimes_{i=1}^d [low_i, high_i]\) প্রদত্ত দৈব (random) বণ্টন অনুসরণ করে।

[3]:

# can be used in case a principal component analysis has been done to derive the uncertainty model, ignored in this example.
A = np.eye(2)
b = np.zeros(2)

# specify the number of qubits that are used to represent the different dimenions of the uncertainty model
num_qubits = [2, 2]

# specify the lower and upper bounds for the different dimension
low = [0, 0]
high = [0.12, 0.24]
mu = [0.12, 0.24]
sigma = 0.01*np.eye(2)

# construct corresponding distribution
bounds = list(zip(low, high))
u = NormalDistribution(num_qubits, mu, sigma, bounds)

নগদ প্রবাহ, পে-অফ ফাংশন এবং সঠিক গড় মান¶

নিম্নলিখিত অংশটিতে আমরা সময়কালে নগদ প্রবাহ সংজ্ঞায়িত করি, ফলাফলের পে-অফ ফাংশন এবং সঠিক গড় মানটি মূল্যায়ন করি।

পে-অফ ফাংশনের জন্য আমরা প্রথমে, প্রথম অর্ডার অনুমান ব্যবহার করি এবং তারপরে European Call Option এর পে-অফ ফাংশনের রৈখিক অংশের মতো একই আনুমানিক পদ্ধতি প্রয়োগ করি।

# estimate real value
cnt = 0
exact_value = 0.0
for x1 in np.linspace(low[0], high[0], pow(2, num_qubits[0])):
    for x2 in np.linspace(low[1], high[1], pow(2, num_qubits[1])):
        prob = u.probabilities[cnt]
        for t in range(len(cf)):
            # evaluate linear approximation of real value w.r.t. interest rates
            exact_value += prob * (cf[t]/pow(1 + b[t], t+1) - (t+1)*cf[t]*np.dot(A[:, t], np.asarray([x1, x2]))/pow(1 + b[t], t+2))
        cnt += 1
print('Exact value:    \t%.4f' % exact_value)

Exact value:            2.1942
