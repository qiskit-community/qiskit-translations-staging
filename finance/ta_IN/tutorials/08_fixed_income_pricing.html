நிலையான வருமான சொத்துக்களின் விலை நிர்ணயம்¶

முன்னுரை¶

தொடர்புடைய வட்டி விகிதங்களை விவரிக்கும் விநியோகங்களை அறிந்து ஒரு நிலையான வருமான சொத்துக்கு விலை கொடுக்க நாங்கள் முயல்கிறோம். பணம் பாய்கிறது :math:`c_t`சொத்து மற்றும் அவை நிகழும் தேதிகள் அறியப்படுகின்றன. மொத்த மதிப்பு :math:`V`சொத்தின் எதிர்பார்ப்பு மதிப்பு:

\[V = \sum_{t=1}^T \frac{c_t}{(1+r_t)^t\]

ஒவ்வொரு பணப்புழக்கமும் தொடர்புடைய வட்டி விகிதத்துடன் பூஜ்ஜிய கூப்பன் பத்திரமாக கருதப்படுகிறது \(r_t\) அதன் முதிர்ச்சியைப் பொறுத்தது. ஒவ்வொன்றிலும் உள்ள நிச்சயமற்ற தன்மையை விநியோக மாதிரியை பயனர் குறிப்பிட வேண்டும் \(r_t\) (தொடர்புடையதாக இருக்கலாம்) அத்துடன் ஒவ்வொரு விநியோகத்தையும் மாதிரியாகப் பயன்படுத்த அவர் விரும்பும் குவிட்களின் எண்ணிக்கை. இந்த எடுத்துக்காட்டில் வட்டி விகிதங்களில் சொத்தின் மதிப்பை முதல் வரிசைக்கு விரிவுபடுத்துகிறோம் \(r_t\). இது சொத்தின் கால அளவைப் படிப்பதற்கு ஒத்துள்ளது. புறநிலை செயல்பாட்டின் தோராயமாக பின்வரும் தாள் பின்வருமாறு: Quantum Risk Analysis. Woerner, Egger. 2018.

நிச்சயமற்ற மாதிரி¶

ஒரு பன்முக இயல்பான சீரற்ற விநியோகத்தை ஏற்ற ஒரு சுற்று தொழிற்சாலையை உருவாக்குகிறோம்:math:d`பரிமாணங்கள் ஒரு குவாண்டம் நிலையில். கொடுக்கப்பட்ட பெட்டியில் விநியோகம் துண்டிக்கப்படுகிறது:math:otimes_{i=1}^d [low_i, high_i]`மற்றும் தனிப்பயனாக்கப்பட்டது \(2^{n_i}\) கட்டம் புள்ளிகள், எங்கே \(n_i பரிமாணத்திற்கு பயன்படுத்தப்படும் குவிட்களின் எண்ணிக்கையைக் குறிக்கிறது:math:\).சுற்று தொழிற்சாலைக்கு ஒத்த ஒற்றுமை ஆபரேட்டர் பின்வருவனவற்றை செயல்படுத்துகிறது:

\[\big|0\rangle_{n_1}\ldots\big|0\rangle_{n_d} \mapsto \big|\psi\rangle = \sum_{i_1=0}^{2^n_-1}\ldots\sum_{i_d=0}^{2^n_-1} \sqrt{p_{i_1,...,i_d}}\big|i_1\rangle_{n_1}\ldots\big|i_d\rangle_{n_d},\]

எங்கே \(p_{i_1, ..., i_d}\) சரியான இடைவெளியில் பொருத்தப்பட்டுள்ளது :math:`[low_j, high_j]`இணைப்பு வரைபடத்தைப் பயன்படுத்தி:

\[\{0, \ldots, 2^{n_{j}}-1\} \ni i_j \mapsto \frac{high_j - low_j}{2^{n_j} - 1} * i_j + low_j \in [low_j, high_j].\]

நிச்சயமற்ற மாதிரியுடன் கூடுதலாக, நாங்கள் ஒரு அஃபைன் வரைபடத்தையும் பயன்படுத்தலாம், எ.கா. ஒரு முக்கிய கூறு பகுப்பாய்வின் விளைவாக. பயன்படுத்தப்படும் வட்டி விகிதங்கள் பின்வருமாறு கொடுக்கப்பட்டது:

\[\vec{r} = A * \vec{x} + b,\]

பகிர்ந்தளித்தல்எங்கே \(\vec{x} \in \otimes_{i=1}^d [low_i, high_i]\) கொடுக்கப்பட்ட சீரற்ற விநியோகத்தைப் பின்பற்றுகிறது.

# can be used in case a principal component analysis has been done to derive the uncertainty model, ignored in this example.
A = np.eye(2)
b = np.zeros(2)

# specify the number of qubits that are used to represent the different dimenions of the uncertainty model
num_qubits = [2, 2]

# specify the lower and upper bounds for the different dimension
low = [0, 0]
high = [0.12, 0.24]
mu = [0.12, 0.24]
sigma = 0.01*np.eye(2)

# construct corresponding distribution
bounds = list(zip(low, high))
u = NormalDistribution(num_qubits, mu, sigma, bounds)

பணப்புழக்கம், செலுத்துதல் செயல்பாடு மற்றும் சரியான எதிர்பார்க்கப்பட்ட மதிப்பு¶

பின்வருவனவற்றில், ஒரு காலகட்டத்திற்கான பணப்புழக்கத்தை நாங்கள் வரையறுக்கிறோம், இதன் விளைவாக செலுத்தும் செயல்பாடு மற்றும் சரியான எதிர்பார்க்கப்படும் மதிப்பை மதிப்பீடு செய்கிறது.

செலுத்துதல் செயல்பாட்டிற்கு, நாங்கள் முதலில் ஒரு முதல் வரிசை தோராயத்தைப் பயன்படுத்துகிறோம், பின்னர் ஐரோப்பிய அழைப்பு விருப்பத்தின் இன் செலுத்துதல் செயல்பாட்டின் நேரியல் பகுதியைப் போலவே அதே தோராய நுட்பத்தையும் பயன்படுத்துகிறோம்.

# estimate real value
cnt = 0
exact_value = 0.0
for x1 in np.linspace(low[0], high[0], pow(2, num_qubits[0])):
    for x2 in np.linspace(low[1], high[1], pow(2, num_qubits[1])):
        prob = u.probabilities[cnt]
        for t in range(len(cf)):
            # evaluate linear approximation of real value w.r.t. interest rates
            exact_value += prob * (cf[t]/pow(1 + b[t], t+1) - (t+1)*cf[t]*np.dot(A[:, t], np.asarray([x1, x2]))/pow(1 + b[t], t+2))
        cnt += 1
print('Exact value:    \t%.4f' % exact_value)
