நிறுவல்¶

Qiskit நிதி அதன் சொந்த Qiskit தொகுப்பைப் பொறுத்தது Qiskit Getting Started Qiskit மற்றும் அதன் ஆதரவு சூழல்கள்/தளங்களுக்கான நிறுவல் விருப்பங்களை விவரித்தல். நீங்கள் அதை முதலில் குறிப்பிட வேண்டும்.Qiskit ஃபைனான்ஸுக்கு குறிப்பிட்ட கூடுதல் நிறுவலில் கவனம் செலுத்தும் தகவல்களை இங்கே பின்பற்றலாம்.

தொடங்குவதற்கான எளிய வழி இங்கே கிஸ்கிட்டிற்கான 'உள்ளூரில் தொடங்கு' என்பதைப் பின்பற்றுவது Qiskit Getting Started

நீங்கள் Qiskit டை நிறுவிய உங்கள் மெய்நிகர் சூழலில் finance ஐ கூடுதல் பட்டியலில் கூடுதல் visualization ஆதரவு எப்படி Qiskit காக நிறுவப்பட்டுள்ளது என்பதைப் போன்றது, அதாவது:

pip install qiskit[finance]

நீங்கள் ஒரு zsh பயனராக இருந்தால் (இது MacOS இன் புதிய பதிப்புகளில் இயல்புநிலை ஷெல்) என்பதை நீங்கள் சுட்டிக்காட்ட வேண்டும் qiskit[finance] மேற்கோள்களில்:

pip install 'qiskit[finance]'

மூலத்திலிருந்து Qiskit ஃபைனான்ஸை நிறுவுவது பைதான் பேக்கேஜ் இன்டெக்ஸ் (பைபிஐ) களஞ்சியத்தில் பதிப்பைப் பயன்படுத்துவதற்குப் பதிலாக, மேம்பாட்டில் உள்ள மிகச் சமீபத்திய மேம்படுத்தப்பட்ட பதிப்பை அணுக அனுமதிக்கிறது. இது Qiskit ஃபைனான்ஸ் குறியீட்டின் சமீபத்திய பதிப்பை மிகவும் திறமையாக ஆய்வு செய்து நீட்டிக்கும் திறனை உங்களுக்கு வழங்கும்.

Qiskit ஃபைனான்ஸ் Qiskit டைப் பொறுத்தது, மேலும் அதன் சமீபத்திய மாற்றங்களுக்கு Qiskit டின் புதிய அல்லது மாற்றப்பட்ட அம்சங்கள் தேவைப்படலாம், நீங்கள் முதலில் Qiskit டைப் பின்பற்ற வேண்டும் "Install from source"`வழிமுறைகள் இங்கே `Qiskit Getting Started

Installing Qiskit Finance from Source

நீங்கள் Qiskit டை நிறுவிய அதே மேம்பாட்டு சூழலைப் பயன்படுத்தி, Qiskit ஃபைனான்ஸை நிறுவத் தயாராக உள்ளீர்கள்.

  1. குளோன் தி Qiskit நிதி களஞ்சியம்.

    git clone https://github.com/Qiskit/qiskit-finance.git

  2. களஞ்சியத்தை குளோனிங் செய்வது qiskit-finance என்ற உள்ளூர் கோப்புறையை உருவாக்குகிறது.

    cd qiskit-finance

  3. நீங்கள் சோதனைகளை இயக்க விரும்பினால் அல்லது சோதனைகள் நீக்கனால், உருவாக்குபவர் தேவைகளை நிறுவவும்.

    pip install -r requirements-dev.txt

  4. qiskit-finance ஐ நிறுவவும்.

    pip install .

நீங்கள் அதை திருத்தக்கூடிய முறையில் நிறுவ விரும்பினால், அதாவது திட்டத்தில் குறியீடு மாற்றங்களை மீண்டும் நிறுவுதல் தேவையில்லை, நீங்கள் இதைச் செய்யலாம்:

pip install -e .
