양자 진폭 추정¶

주어진 연산자 \(\mathcal{A}\) 가 다음과 같이 작용할 때

\[\mathcal{A}|0\rangle = \sqrt{1 - a}|\Psi_0\rangle + \sqrt{a}|\Psi_1\rangle\]

양자 진폭 추정(Quantum Amplitude Estimation, QAE)은 상태 \(|\Psi_1\rangle\) 의 진폭 \(a\) 의 추정치를 찾는 작업이다:

\[a = |\langle\Psi_1 | \Psi_1\rangle|^2.\]

이 작업은 2000년에 Brassard et al. [1] 에 의해 먼저 연구되었으며, 해당 알고리즘은 다음의 Grover 연산자

\[\mathcal{Q} = \mathcal{A}\mathcal{S}_0\mathcal{A}^\dagger\mathcal{S}_{\Psi_1}\]

(여기서 \(\mathcal{S}_0\) 와 \(\mathcal{S}_{\Psi_1}\) 는 각각 \(|0\rangle\) 과 \(|\Psi_1\rangle\) 상태의 반사 연산자이다) 와 페이즈 추정의 조합을 사용한다. 그런데 Qiskit에서 AmplitudeEstimation 로 불리는 이 알고리즘은 큰 회로를 필요로 하고 계산량이 많다. 그러므로 QAE의 다른 변형들이 제안되었고, 이 지침서에서 간단한 예제로 소개할 것이다.

이 예에서는 \(\mathcal{A}\) 가 (모르는 상태로 가정된) 성공 확률 \(p\) 를 갖는 Bernoulli 임의 변수(random variable)을 기술한다:

\[\mathcal{A}|0\rangle = \sqrt{1 - p}|0\rangle + \sqrt{p}|1\rangle.\]

양자 컴퓨터에서는 이 연산자를 단일 큐비트의 \(Y\)-축을 중심으로 한 회전으로 만들 수 있다.

\[\mathcal{A} = R_Y(\theta_p), \theta_p = 2\sin^{-1}(\sqrt{p}).\]

이 경우에서 Grover 연산자는 특히 간단하다

\[\mathcal{Q} = R_Y(2\theta_p),\]

제곱도 매우 간단하게 계산할 수 있다: \(\mathcal{Q}^k = R_Y(2k\theta_p)\).

추정할 확률로써 \(p = 0.2\) 를 고정시킬 것이다.

[1]:

p = 0.2

Now we can define circuits for \(\mathcal{A}\) and \(\mathcal{Q}\).

[2]:

import numpy as np
from qiskit.circuit import QuantumCircuit

class BernoulliA(QuantumCircuit):
    """A circuit representing the Bernoulli A operator."""

    def __init__(self, probability):
        super().__init__(1)  # circuit on 1 qubit

        theta_p = 2 * np.arcsin(np.sqrt(probability))
        self.ry(theta_p, 0)


class BernoulliQ(QuantumCircuit):
    """A circuit representing the Bernoulli Q operator."""

    def __init__(self, probability):
        super().__init__(1)  # circuit on 1 qubit

        self._theta_p = 2 * np.arcsin(np.sqrt(probability))
        self.ry(2 * self._theta_p, 0)

    def power(self, k):
        # implement the efficient power of Q
        q_k = QuantumCircuit(1)
        q_k.ry(2 * k * self._theta_p, 0)
        return q_k

[3]:

A = BernoulliA(p)
Q = BernoulliQ(p)

Qiskit의 진폭 추정 워크플로우¶

Qiskit은 AmplitudeEstimator 인터페이스에서 파생되는 여러 QAE 알고리즘들을 구현한다. 초기설정자에서는 알고리즘의 특정 설정을 지정하고, 작업을 전부 수행하는 estimate 메서드는 EstimationProblem 을 입력 받아 AmplitudeEstimationResult 객체를 반환한다. 모든 QAE의 변형들은 같은 인터페이스를 따르기 때문에 같은 문제 인스턴스를 풀 때 모두 사용할 수 있다.

다음으로 모든 QAE 알고리즘들을 실행할 것이다. 그러려면 먼저 (간단한 예제로서 \(|1\rangle\) 인) \(|\Psi_1\rangle\) 상태를 식별하는 방법뿐만 아니라 \(\mathcal{A}\) 와 \(\mathcal{Q}\) 연산자들을 포함할 추정 문제(estimation problem)을 정의한다.

[4]:

from qiskit.algorithms import EstimationProblem

problem = EstimationProblem(
    state_preparation=A,  # A operator
    grover_operator=Q,  # Q operator
    objective_qubits=[0],  # the "good" state Psi1 is identified as measuring |1> in qubit 0
)

회로를 실행하고자 Qiskit의 상태벡터(statevector) 시뮬레이터를 사용할 것이다.

[5]:

from qiskit import BasicAer
from qiskit.utils import QuantumInstance

backend = BasicAer.get_backend('statevector_simulator')
quantum_instance = QuantumInstance(backend)

정준 AE (Canonical AE)¶

이제 Brassard et al. [1] 에서 구현된 원래 QAE 구현으로 풀어보자.

[6]:

from qiskit.algorithms import AmplitudeEstimation

ae = AmplitudeEstimation(
    num_eval_qubits=3,  # the number of evaluation qubits specifies circuit width and accuracy
    quantum_instance=quantum_instance
)

정의된 알고리즘에서는 estimate 메서드를 호출하고 문제를 제공하여 풀 수 있다.

[7]:

ae_result = ae.estimate(problem)

추정치는 estimation 키를 통해 얻을 수 있다.

[8]:

print(ae_result.estimation)

0.1464466

목표였던 \(p=0.2\) 에 대해 별로 좋은 추정이 아니라는 것을 보았다! 그것은 정준 AE가 계산하는 큐비트 개수로 지정된 이산 그리드에 제한되기 때문이다:

[9]:

import matplotlib.pyplot as plt

# plot estimated values
gridpoints = list(ae_result.samples.keys())
probabilities = list(ae_result.samples.values())

plt.bar(gridpoints, probabilities, width=0.5/len(probabilities))
plt.axvline(p, color='r', ls='--')

추정치를 개선하고자 측정 확률을 보간(interpolate) 하고 확률 분포를 생성하는 최우 추정치(maximum likelihood estimator)를 계산한다.

[10]:

print('Interpolated MLE estimator:', ae_result.mle)

Interpolated MLE estimator: 0.19999999390907777

AE가 실행하는 회로를 살펴볼 수 있다:

반복 진폭 추정 (Iterative Amplitude Estimation)¶

[2] 를 보라.

[13]:

from qiskit.algorithms import IterativeAmplitudeEstimation

iae = IterativeAmplitudeEstimation(
    epsilon_target=0.01,  # target accuracy
    alpha=0.05,  # width of the confidence interval
    quantum_instance=quantum_instance
)
iae_result = iae.estimate(problem)

print('Estimate:', iae_result.estimation)

Estimate: 0.19999999999999998

이 회로는 오직 Grover 지수로만 구성되고 훨씬 단순하다!

[14]:

iae_circuit = iae.construct_circuit(problem, k=3)
iae_circuit.draw('mpl', style='iqx')

[3] 을 보라.

from qiskit.algorithms import MaximumLikelihoodAmplitudeEstimation

mlae = MaximumLikelihoodAmplitudeEstimation(

mlae_result = mlae.estimate(problem)

print('Estimate:', mlae_result.estimation)
