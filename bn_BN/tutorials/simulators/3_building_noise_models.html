ত্রুটি নকশা নির্মাণ¶

ভূমিকা¶

এই নোটবুকে দেখানো হয়েছে কিভাবে Qiskit এয়ারের noise মডিউল ব্যবহার করে ত্রুটিপূর্ণ সিমুলেশনের জন্য বিশেষায়িত (কাস্টম) ত্রুটি নকশা বানাতে হয়।

Qiskit এয়ার ত্রুটি নকশা¶

Qiskit এয়ারের noise মডিউলে Python -এর বিভিন্ন ক্লাস বানানো আছে যেগুলো ব্যবহার করে সিমুলেশনের জন্য প্রয়োজনমতো ত্রুটি নকশা বানানো হয়। এদের মধ্যে প্রধান তিনটি ক্লাস হল:

  1. NoiseModel ক্লাসে ত্রুটিপূর্ণ সিমুলেশনের জন্য একটা ত্রুটি নকশা সঞ্চয় করে রাখে।

  2. QuantumError ক্লাসে সিপিটিপি (CPTP) যুক্তিবর্তনী (গেইট) ত্রুটির বিবরণ দেয়। এগুলো নিম্নলিখিত ক্ষেত্রে ব্যবহার করা যায়:

    • gate বা reset নির্দেশনার পর

    • পরিমাপের আগের নির্দেশনা।

  3. ReadoutError ক্লাস ধ্রুপদী রিডআউট ত্রুটির বিবরণ দেয়।

কোয়ান্টাম ত্রুটি¶

QuantumError অবজেক্ট এ সরাসরি হাত না দেওয়ার বদলে আমরা অন্য সাহায্যকারী ফাংশন ব্যবহার করতে পারি যাতে স্বয়ংক্রিয়ভাবে একটা নির্দিষ্ট রকমের পরামিতিক (প্যারামিটারাইজড) কোয়ান্টাম ত্রুটি তৈরী হয়। এগুলো noise মডিউলে পাওয়া যায় এবং কোয়ান্টাম কম্পিউটিং গবেষণার বহুরকমের ত্রুটির ফাংশন এখানে বানানো আছে। ফাংশন এর নাম ও তাদের ব্যবহৃত ত্রুটির নাম হলো:

সাধারণ ত্রুটি ফাংশন

বিবরণ

kraus_error

একটা সাধারণ n-কিউবিটসম্পন্ন সিপিটিপি (CPTP) ত্রুটি চ্যানেল আমাদের একটা ক্রাউস ম্যাট্রিক্সের \([K_0, ...]\) তালিকা দেয়।

mixed_unitary_error

একটা n-কিউবিটসম্পন্ন মিশ্র ঐকিক (ইউনিটারি) ত্রুটি একটা ঐকিক (ইউনিটারি) ম্যাট্রিক্স ও সম্ভাবনার \([(U_0, p_0),...]\) তালিকা রূপে দেওয়া হয়।

coherent_unitary_error

একটা n-কিউবিট সুসংহত ঐকিক (ইউনিটারি) ত্রুটি একটা ঐকিক (ইউনিটারি) ম্যাট্রিক্স \(U\) রূপে দেওয়া হয়।

pauli_error

একটা n-কিউবিট পাউলি ত্রুটি চ্যানেল (মিশ্র ঐকিক (ইউনিটারি)) একটা পাউলির এবং সম্ভাবনার \([(P_0, p_0),...]\) তালিকারূপে দেওয়া হয়।

depolarizing_error

একটা n-কিউবিট ডি-পোলারাইজিঙ ত্রুটি চ্যানেল যার প্যারামিটার হল একটা ডি-পোলারাইজেশন সম্ভাবনা \(p\)।

reset_error

একটা একক কিউবিট রিসেট বা পুনঃস্থাপন ত্রুটি যার প্যারামিটার হলো \(|0\rangle\), \(|1\rangle\) অবস্থায় রিসেট বা পুনঃস্থাপন হওয়ার সম্ভাবনা \(p_0, p_1\)।

thermal_relaxation_error

একটা একক কিউবিট তাপীয় শিথিলকরণ (রিলাক্সেশন) চ্যানেল যার প্যারামিটার হল শিথিলকরণ (রিলাক্সেশন) সময় ধ্রুবক \(T_1\), \(T_2\), যুক্তিবর্তনী (গেইট) সময় \(t\) এবং উত্তেজিত দশা তাপীয় সংখ্যাবিন্যাস (থার্মাল পপুলেশন) \(p_1\)।

phase_amplitude_damping_error

একটা একক কিউবিট সাধারণ মিলিত দশা এবং বিস্তার অবনমনকারী (ড্যাম্পিং) ত্রুটি চ্যানেল বানাতে হলে একটা বিস্তার অবনমন (ড্যাম্পিং) প্যারামিটার \(\lambda\), একটা দশা অবনমন (ড্যাম্পিং) প্যারামিটার \(\gamma\) এবং একটা উত্তেজিত দশা তাপীয় সংখ্যা বা সংখ্যাবিন্যাস (থার্মাল পপুলেশন) \(p_1\) প্রয়োজন।

amplitude_damping_error

একটা একক কিউবিট সাধারণ বিস্তার অবনমনকারী (ড্যাম্পিং) ত্রুটি চ্যানেল বানাতে হলে একটা বিস্তার অবনমন (ড্যাম্পিং) প্যারামিটার \(\lambda\) এবং একটা উত্তেজিত দশা তাপীয় সংখ্যা বা সংখ্যাবিন্যাস (থার্মাল পপুলেশন) \(p_1\) প্রয়োজন।

phase_damping_error

একটা একক কিউবিট দশা অবনমনকারী (ড্যাম্পিং) ত্রুটি চ্যানেল বানাতে হলে দশা অবনমন (ড্যাম্পিং) প্যারামিটার \(\gamma\) দিতে হবে

কোয়ান্টাম ত্রুটির সংমিশ্রণ¶

QuantumError দৃষ্টান্তগুলোর (ইনস্ট্যান্স) উপর কম্পোসিশন, টেনসর গুনফল (প্রোডাক্ট) এবং টেনসর বিস্তৃতি (এক্সপ্যানশন) (বিপরীত ক্রমের টেনসর গুনফল (প্রোডাক্ট)) করে নতুন QuantumErrors তৈরী করা যায়। উদাহরণস্বরূপ:

  • গঠন: \(\cal{E}(\rho)=\cal{E_2}(\cal{E_1}(\rho))\) যেখানে error = error1.compose(error2)

  • টেনসর গুণফল (প্রোডাক্ট): \(\cal{E}(\rho) =(\cal{E_1}\otimes\cal{E_2})(\rho)\) যেখানে error error1.tensor(error2)

  • গুনফলের প্রসারণ: \(\cal{E}(\rho) =(\cal{E_2}\otimes\cal{E_1})(\rho)\) যেখানে error error1.expand(error2)

রিডআউট ত্রুটি¶

ধ্রুপদী রিডআউট ত্রুটি হলো একটা মমান বন্টনের সম্ভাবনা তালিকা \(P(A|B)\):

  • \(A\) হলো ধ্রুপদী বিটের সঞ্চিত মান

  • \(B\) হলো পরিমাপ থেকে পাওয়া বিটের আসল মান

উদাহরণস্বরূপ একটা কিউবিটের ক্ষেত্রে: $ P(A|B) = [P(A|0), P(A|1)]$।

[8]:

# Measurement miss-assignement probabilities
p0given1 = 0.1
p1given0 = 0.05

ReadoutError([[1 - p1given0, p1given0], [p0given1, 1 - p0given1]])

[8]:

ReadoutError([[0.95 0.05]
 [0.1  0.9 ]])

রিডআউট ত্রুটিও কোয়ান্টাম ত্রুটির মতো compose, tensor এবং expand ব্যবহার করে একত্রিত করা যায়।

ত্রুটি নকশায় ত্রুটি যোগ করা¶

ত্রুটি নকশায় কোয়ান্টাম ত্রুটি যোগ করার সময় আমাদের বলতে হবে কোন ধরণের নির্দেশে দেওয়ায় এই ত্রুটি কাজ করবে এবং কোন কোন কিউবিটের উপর কাজ করবে। কোয়ান্টাম ত্রুটির ক্ষেত্রে তিনটি ঘটনা হতে পারে:

  1. সমস্ত কিউবিট কোয়ান্টাম ত্রুটি

  2. নির্দিষ্ট কিউবিট কোয়ান্টাম ত্রুটি

  3. অ-স্থানীয় কোয়ান্টাম ত্রুটি

সমস্ত কিউবিট কোয়ান্টাম ত্রুটি¶

এটা একই ত্রুটি একটা নির্দিষ্ট নির্দেশের সব ক্ষেত্রে সব কিউবিটের উপর কাজ করবে।

এটা যোগ করতে হলে noise_model.add_all_qubit_quantum_error(error, instructions):

[9]:

# Create an empty noise model
noise_model = NoiseModel()

# Add depolarizing error to all single qubit u1, u2, u3 gates
error = depolarizing_error(0.05, 1)
noise_model.add_all_qubit_quantum_error(error, ['u1', 'u2', 'u3'])

# Print noise model info
print(noise_model)

  All-qubits errors: ['u1', 'u2', 'u3']

নির্দিষ্ট কিউবিট কোয়ান্টাম ত্রুটি¶

এটা কোনো নির্দেশনার সকল ক্ষেত্রে কাজ করে, কিন্তু কোন কোন কিউবিটের উপর কাজ করবে সেটা প্রদত্ত কিউবিটের তালিকার উপর নির্ভর করে। মনে রাখতে হবে কিউবিটের ক্রম ঠিক রাখতে হবে, অর্থাৎ একটা ২-কিউবিট যুক্তিবর্তনীতে (গেইট) [০, ১] কিউবিটে ত্রুটি যোগ করা এবং [১, ০] কিউবিটে ত্রুটি যোগ করা এক নয়। উদাহরণস্বরূপ:

এটা যোগ করতে হলে noise_model.add_quantum_error(error, instructions, qubits):

[10]:

# Create an empty noise model
noise_model = NoiseModel()

# Add depolarizing error to all single qubit u1, u2, u3 gates on qubit 0 only
error = depolarizing_error(0.05, 1)
noise_model.add_quantum_error(error, ['u1', 'u2', 'u3'], [0])

# Print noise model info
print(noise_model)

অ-স্থানীয় কিউবিট কোয়ান্টাম ত্রুটি¶

এক্ষেত্রে ত্রুটিটা একটা নির্দিষ্ট নির্দেশনা একটা নির্দিষ্ট যুক্তিবর্তনী (গেইট) কিউবিটের উপর কাজ করার পর একটা ত্রুটিপূর্ণ কিউবিটের সেটের উপর কাজ করে।

এটা যোগ করতে হলে noise_model.add_nonlocal_quantum_error(error, instructions, instr_qubits, error_qubits):

[11]:

# Create an empty noise model

# Add depolarizing error on qubit 2 forall single qubit u1, u2, u3 gates on qubit 0
error = depolarizing_error(0.05, 1)
noise_model.add_nonlocal_quantum_error(error, ['u1', 'u2', 'u3'], [0], [2])

# Print noise model info
print(noise_model)

একটা ত্রুটিপূর্ণ সিমুলেশন একটা ত্রুটি নকশার সাথে চালানো¶

AerSimulator(noise_model=noise_model) নির্দেশটি প্রদত্ত ত্রুটি নকশার একটা সিমুলেটর বানায়। একইসঙ্গে এটা সিমুলেটরের বেসিস (ভিত্তি) যুক্তিবর্তনীর (গেইট) বদলে ত্রুটি নকশার যুক্তিবর্তনী (গেইট) গুলো কাজে লাগায়।

উদাহরণ ১: সাধারণ বিট অদলবদল ত্রুটি নকশা¶

কোয়ান্টাম তথ্য তত্ত্ব গবেষণার একটা সাধারণ ত্রুটি নকশাকে উদাহরণরূপে নেওয়া যাক:

  • যখন একক কিউবিট যুক্তিবর্তনীতে (গেইট) ব্যবহার হয়, কিউবিটের মান বদলের সম্ভাবনা p_gate1।

  • ২-কিউবিটসম্পন্ন যুক্তিবর্তনীতে (গেইট) প্রয়োগের সময় প্রত্যেক কিউবিটে একক কিউবিট ত্রুটি প্রয়োগ করা হয়।

  • যখন একটা কিউবিট p_reset সম্ভাবনায় ০ এর বদলে ১ এ রিসেট বা পুনঃস্থাপন করতে হয়।

  • কিউবিট পরিমাপকালীন কিউবিটের অবস্থা p_meas সম্ভাবনায় পরিবর্তন করা হয়।

p_reset = 0.03

উদাহরণ ২: T1/T2 তাপীয় শিথিলকরণ (রিলাক্সেশন)¶

এবার তাপীয় শিথিলকরণ (রিলাক্সেশন) উপর নির্ভর করে কিউবিট কর্মপরিবেশে একটা বাস্তব ত্রুটি নকশা নিয়ে ভাবা যাক: * প্রত্যেক কিউবিটে প্যারামিটাররূপে একটা তাপীয় শিথিলকরণ (রিলাক্সেশন) সময় ধ্রুবক \(T_1\) এবং একটা অবসন্ন সময় ধ্রুবক \(T_2\) দিতে হবে। * মনে রাখতে হবে \(T_2 \le 2 T_1\)। * নির্দেশনায় ত্রুটির হার যুক্তিবর্তনী (গেইট) সময় ও কিউবিট \(T_1\), \(T_2\) মানের উপর নির্ভরশীল।

[16]:

# T1 and T2 values for qubits 0-3
T1s = np.random.normal(50e3, 10e3, 4) # Sampled from normal distribution mean 50 microsec
T2s = np.random.normal(70e3, 10e3, 4)  # Sampled from normal distribution mean 50 microsec

# Truncate random T2s <= T1s
T2s = np.array([min(T2s[j], 2 * T1s[j]) for j in range(4)])

# Instruction times (in nanoseconds)
time_u1 = 0   # virtual gate
time_u2 = 50  # (single X90 pulse)
time_u3 = 100 # (two X90 pulses)
time_cx = 300
time_reset = 1000  # 1 microsecond
time_measure = 1000 # 1 microsecond

# QuantumError objects
errors_reset = [thermal_relaxation_error(t1, t2, time_reset)
                for t1, t2 in zip(T1s, T2s)]
errors_measure = [thermal_relaxation_error(t1, t2, time_measure)
                  for t1, t2 in zip(T1s, T2s)]
errors_u1  = [thermal_relaxation_error(t1, t2, time_u1)
              for t1, t2 in zip(T1s, T2s)]
errors_u2  = [thermal_relaxation_error(t1, t2, time_u2)
              for t1, t2 in zip(T1s, T2s)]
errors_u3  = [thermal_relaxation_error(t1, t2, time_u3)
              for t1, t2 in zip(T1s, T2s)]
errors_cx = [[thermal_relaxation_error(t1a, t2a, time_cx).expand(
             thermal_relaxation_error(t1b, t2b, time_cx))
              for t1a, t2a in zip(T1s, T2s)]
               for t1b, t2b in zip(T1s, T2s)]

# Add errors to noise model
noise_thermal = NoiseModel()
for j in range(4):
    noise_thermal.add_quantum_error(errors_reset[j], "reset", [j])
    noise_thermal.add_quantum_error(errors_measure[j], "measure", [j])
    noise_thermal.add_quantum_error(errors_u1[j], "u1", [j])
    noise_thermal.add_quantum_error(errors_u2[j], "u2", [j])
    noise_thermal.add_quantum_error(errors_u3[j], "u3", [j])
    for k in range(4):
        noise_thermal.add_quantum_error(errors_cx[j][k], "cx", [j, k])

print(noise_thermal)
