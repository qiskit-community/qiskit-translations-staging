Qiskit অ্যালগরিদম (ধারাক্রম) পরিযান নির্দেশনা¶

অ্যাপ্লিকেশনের পুনর্গঠন

Qiskit 0.25.0 রিলিজ বা প্রকাশে অ্যাপ্লিকেশন এবং অ্যালগরিদমের পুনর্গঠন হয়েছে। পূর্বে Qiskit এর একক অ্যাপ্লিকেশন এবং অ্যালগরিদম মডিউল ছিল Qiskit অ্যাকোয়া, এখন সেটি অনুকূলকরণ (অপটিমাইজেশন), ব্যবস্থাপনাবিদ্যা, মেশিন লার্নিং এবং প্রকৃতি বিজ্ঞানের (পদার্থবিজ্ঞান এবং রসায়ন সহ) জন্য উৎসর্গীকৃত অ্যাপ্লিকেশন মডিউলে বিভক্ত। মূল অ্যালগরিদমগুলি এবং opflow( অপফ্লো) অপারেটরের কার্যকারিতা Qiskit টেরা তে স্থানান্তরিত করা হয়েছে।

ধারাক্রম (অ্যালোগরিদম) ইন্টারফেস

পুনর্গঠন ছাড়াও, সমস্ত অ্যালোগরিদম একটি নতুন সমন্বিত দৃষ্টান্ত অনুসরণ করে: অ্যালগোরিদমগুলি তাদের সমাধান করা সমস্যার অনুসারে শ্রেণিবদ্ধ করা হয় এবং একই অ্যাপ্লিকেশন বর্গের মধ্যে একটি সমস্যা সমাধানের জন্য অ্যালোগরিদমগুলি আন্তঃ বিনিময়যোগ্য হিসাবে ব্যবহার করা যেতে পারে। এর অর্থ হ’ল, এখন অ্যালগরিদম দৃষ্টান্ত (ইনস্ট্যান্স) গুলি যে সমস্যাটি সমাধান করে তা থেকে ডিকাপ্লিং করা হয়। আমরা এটি একটি ফ্লোচার্টে সংক্ষিপ্ত করতে পারি:

উদাহরণস্বরূপ ভেরিয়েশনাল কোয়ান্টাম আইজেনসলভার (ভিকিউই) VQE হল MinimumEigensolver যেহেতু এটি একটি অপারেটরের সর্বনিম্ন স্বকীয়মান (আইগেনভ্যালু) গণনা করে। এখানে সমস্যাটি অপারেটরের সাথে জড়িত, যার স্বকীয়মান (আইগেনভ্যালু) আমাদের দরকার, যখন পরিবর্তনশীল (ভ্যারিয়েশনাল) অন্সাৎজ বর্তনী (সার্কিট) এবং প্রথাগত (ক্লাসিক্যাল) অনুকূলায়ক এই নীতিগুলি হল ধারাক্রম (অ্যালোগরিদম) এর নীতি। অর্থাৎ `` VQE`` এর নিম্নলিখিত কাঠামো রয়েছে

vqe = VQE(ansatz, optimizer)
result = vqe.compute_minimum_eigenvalue(operator)

আমরা VQE``কে অন্য যেকোন ধারাক্রম (অ্যালোগরিদম)এর সাথে বিনিময় করতে পারি যা অপারেটরের স্বকীয়মান (আইগেনভ্যালু) গণনা করতে ``MinimumEigensolver ইন্টারফেস প্রয়োগ করে, যেমন:

numpy_based = NumPyMinimumEigensolver()
classical_reference = numpy_based.compute_minimum_eigenvalue(operator)

এটি আপনাকে সহজেই বিভিন্ন অ্যালগোরিদমের মধ্যে স্থান পরিবর্তন করতে, প্রথাগত (ক্লাসিক্যাল) তথ্যসূত্রের (রেফারেন্স) পরীক্ষা করতে এবং আপনার নিজস্ব রূপায়ণ (ইমপ্লিমেন্টেশন) \(-\) যোগান করতে দেয়: আপনাকে কেবল বিদ্যমান ইন্টারফেসটি প্রয়োগ করতে হবে।

এই নোটবুকটি নতুন কাঠামোর সাথে Qiskit অ্যাকোয়া ব্যবহার করে আপনার বর্তমান কোড পরিবর্তন করার সুবিধার্থে মাইগ্রেশন গাইড হিসাবে কাজ করে।

আমরা এই নোটবুকের জন্য ডেপ্রিকেশন (অপসারণ) সতর্কবার্তাটি অক্ষম করছি যাতে একটি অবজেক্টকে তাৎক্ষনিক করার সময় আপনি qiskit.aqua কিছু দেখতে না পান। তবে লক্ষনীয়, যে পুরো প্যাকেজটি ডেপ্রিকেটেড হয়ে গেছে এবং নিম্নলিখিতগুলির মতো একটি সতর্কতা প্রেরণ করবে:

গ্রোভার¶

সারসংক্ষেপ¶

grover = Grover(oracle_settings, algorithm_settings)

অভিপ্রয়াণ প্রদর্শক¶

ওরাকলগুলির জন্য ভালো মান বা অবস্থা নির্ধারণে বর্তনী (সার্কিট) এবং is_good_state ক্রিয়া সরবরাহ করা হয়

[10]:

from qiskit.circuit import QuantumCircuit

oracle = QuantumCircuit(2)
oracle.cz(0, 1)

def is_good_state(bitstr):
    return sum(map(int, bitstr)) == 2

পূর্বে:

[11]:

from qiskit.aqua.algorithms import Grover

grover = Grover(oracle, is_good_state, quantum_instance=aqua_qinstance)
result = grover.run()
print('Top measurement:', result.top_measurement)

Top measurement: 11

নতুন:

[12]:

from qiskit.algorithms import Grover, AmplificationProblem

problem = AmplificationProblem(oracle=oracle, is_good_state=is_good_state)
grover = Grover(quantum_instance=qinstance)
result = grover.amplify(problem)
print('Top measurement:', result.top_measurement)

Top measurement: 11

যেহেতু আমরা QuantumCircuit শ্রেণীকে মৌলিক ভিত্তি হিসেবে ব্যবহার করতে সব অ্যালোগরিদমকে প্রবাহিত (স্ট্রিমলাইন) করছি, qiskit.aqua.compontents.Oracle ব্যবহার করে ওরাকেল সংজ্ঞায়িত করা ডেপ্রিকেটেড। LogicalExpressionOracle ব্যবহারের পরিবর্তে এখন বর্তনী (সার্কিট) লাইব্রেরির PhaseOracle বর্তনী (সার্কিট) ব্যবহার করা যায়।

পূর্বে:

[13]:

from qiskit.aqua.components.oracles import LogicalExpressionOracle
from qiskit.aqua.algorithms import Grover

oracle = LogicalExpressionOracle('x & ~y')
grover = Grover(oracle, quantum_instance=aqua_qinstance)
result = grover.run()
print('Top measurement:', result.top_measurement)

Top measurement: 01

নতুন:

[14]:

from qiskit.circuit.library import PhaseOracle
from qiskit.algorithms import Grover, AmplificationProblem

oracle = PhaseOracle('x & ~y')
problem = AmplificationProblem(oracle=oracle, is_good_state=oracle.evaluate_bitstring)
grover = Grover(quantum_instance=qinstance)
result = grover.amplify(problem)
print('Top measurement:', result.top_measurement)

Top measurement: 01

qiskit.aqua.components.oracles.TruthTableOracle টি এখনো পোর্ট করা হয়নি, কিন্তু তার ধর্ম qiskit.circuit.classicalfunction মডিউল দিয়ে সহজেই অর্জন করা যায়, গ্রোভার ধারাক্রম (অ্যালোগরিদম)-এর পাঠসমূহ (টিউটোরিয়ালস)টি দেখুন।

power = 2
grover.construct_circuit(problem, power).draw('mpl', style='iqx')

বিস্তার অনুমান¶

সারসংক্ষেপ¶

সমস্ত বিস্তার অনুমান অ্যালোগরিদমের জন্য * AmplitudeEstimation * IterativeAmplitudeEstimation * MaximumLikelihoodAmplitudeEstimation, এবং * FasterAmplitudeEstimation

qae = AmplitudeEstimation(algorithm_settings, estimation_settings)

সমস্যা /ওরাকল বিন্যাস এবং অ্যালগরিদম বিন্যাসে বিভক্ত করতে

qae = AmplitudeEstimation(algorithm_settings)
problem = EstimationProblem(oracle_settings)
result = qae.amplify(problem)

অভিপ্রয়াণ প্রদর্শক¶

এখানে আমরা একটি কিউবিটের \(|1\rangle\) পরিমাপের সম্ভাবনা অনুমান করতে চাই। যদি মান বা অবস্থার প্রস্তুতিকে বর্তনী (সার্কিট) হিসেবে প্রেরণ করা হয়

[16]:

import numpy as np

probability = 0.25
rotation_angle = 2 * np.arcsin(np.sqrt(probability))

state_preparation = QuantumCircuit(1)
state_preparation.ry(rotation_angle, 0)

objective_qubits = [0]  # the good states are identified by qubit 0 being in state |1>

print('Target probability:', probability)

state_preparation.draw(output='mpl', style='iqx')

Target probability: 0.25

[16]:

পূর্বে:

[17]:

from qiskit.aqua.algorithms import AmplitudeEstimation

# instantiate the algorithm and passing the problem instance
ae = AmplitudeEstimation(3, state_preparation, quantum_instance=aqua_qinstance)

# run the algorithm
result = ae.run()

# print the results
print('Grid-based estimate:', result.estimation)
print('Improved continuous estimate:', result.mle)

Grid-based estimate: 0.1464466
Improved continuous estimate: 0.2499999956526605

এখন:

HHL¶

সারসংক্ষেপ¶

এইচএইচএলকে (HHL) একটি স্বজ্ঞাত ইন্টারফেসের অনুমতি দিতে এবং ফলাফলের একটি দক্ষ, সার্কিট-ভিত্তিক উপস্থাপনা ফিরিয়ে দেওয়ার জন্য পুরোপুরি রিফ্যাক্টর করা হয়েছে।

অভিপ্রয়াণ প্রদর্শক¶

ধরুন আমরা নিম্নলিখিত একঘাত (লিনিয়ার) সিস্টেমটি সমাধান করতে চাই

\[\begin{split}\begin{pmatrix} 1 & -1/3 \\ -1/3 & 1 \\ \end{pmatrix} \vec x = \begin{pmatrix} 1 \\ 0 \end{pmatrix}\end{split}\]
[42]:

import numpy as np

matrix = np.array([[1, -1/3], [-1/3, 1]])
vector = np.array([1, 0])

পূর্বে:

[43]:

from qiskit.circuit.library import QFT
from qiskit.aqua.algorithms import HHL
from qiskit.aqua.components.eigs import EigsQPE
from qiskit.aqua.components.reciprocals import LookupRotation
from qiskit.aqua.components.initial_states import Custom
from qiskit.aqua.operators import MatrixOperator

def create_eigs(matrix, num_auxiliary, num_time_slices, negative_evals):
    ne_qfts = [None, None]
    if negative_evals:
        num_auxiliary += 1
        ne_qfts = [QFT(num_auxiliary - 1), QFT(num_auxiliary - 1).inverse()]
    return EigsQPE(MatrixOperator(matrix=matrix),
                   QFT(num_auxiliary).inverse(),
                   num_time_slices=num_time_slices,
                   num_ancillae=num_auxiliary,
                   expansion_mode='suzuki',
                   expansion_order=2,
                   evo_time=None,
                   negative_evals=negative_evals,
                   ne_qfts=ne_qfts)

orig_size = len(vector)

matrix, vector, truncate_powerdim, truncate_hermitian = HHL.matrix_resize(matrix, vector)

# Initialize eigenvalue finding module
eigs = create_eigs(matrix, 3, 50, False)
num_q, num_a = eigs.get_register_sizes()

# Initialize initial state module
init_state = Custom(num_q, state_vector=vector)

# Initialize reciprocal rotation module
reci = LookupRotation(negative_evals=eigs._negative_evals, evo_time=eigs._evo_time)
algo = HHL(matrix, vector, truncate_powerdim, truncate_hermitian, eigs,
           init_state, reci, num_q, num_a, orig_size)

result = algo.run(aqua_qinstance)
print(result.solution)

[ 0.66575607+4.96430258e-13j -0.38561455-2.84524449e-13j]

দশা (ফেজ) অনুমান¶

সারসংক্ষেপ¶

দশা (ফেজ) অনুমান সম্পূর্ণরূপে রিফ্যাক্টর করা হয়েছে এবং হ্যামিল্টোনিয়ানের স্বকীয়মান (আইগেনভ্যালু) গণনা করতে ব্যবহৃত একটি মাত্র qiskit.aqua.algorithms.QPE ক্লাসের পরিবর্তে, আমাদের এখন দুটি পৃথক রূপায়ণ (ইমপ্লিমেন্টেশন) রয়েছে: HamiltonianPhaseEstimation পুরানো QPE এর ভূমিকা গ্রহণ করে এবং পাঠ্যপুস্তক পর্যায়ের অনুমানের জন্য একটি নতুন PhaseEstimation ধারাক্রম (অ্যালোগরিদম)।

পুনরাবৃত্তিমূলক দশা (ফেজ) অনুমান, qiskit.aqua.algorithms.IQPE এখনও প্রতিস্থাপিত হয়নি কিন্তু শীঘ্রই করা হবে।

অভিপ্রয়াণ প্রদর্শক¶

স্বকীয়মান (আইগেনভ্যালু) খোঁজার সমস্যা বিবেচনা করা যাক

\[H = 0.5 X + Y + Z\]

ইনপুট মান বা অবস্থা \(|0\rangle\).

[47]:

state_in = np.array([1, 0])

পূর্বে:

[48]:

from qiskit.circuit.library import QFT
from qiskit.aqua.algorithms import QPE
from qiskit.aqua.components.initial_states import Custom
from qiskit.aqua.operators import I, X, Y, Z

n_ancillae = 5
num_time_slices = 1

op = 0.5 * X + Y + Z

state_preparation = Custom(op.num_qubits, state_vector=state_in)
iqft = QFT(n_ancillae, do_swaps=False).inverse().reverse_bits()

qpe = QPE(op, state_preparation, iqft, num_time_slices, n_ancillae,
          expansion_mode='trotter',
          shallow_circuit_concat=True)


result = qpe.run(aqua_qinstance)

print(result.eigenvalue)

0.78125

import numpy as np
from qiskit.opflow import I, X, Y, Z, PauliTrotterEvolution, StateFn

hamiltonian = (X ^ X) + (Z ^ Z) + 1j * (Y ^ Y)

observable = X ^ I

initial_state = StateFn(np.ones(4) / 2)

evo_time = 2
num_time_slices = 10

# get the evolution operator
evolved_hamiltonian = (evo_time * hamiltonian).exp_i()

evo = PauliTrotterEvolution(reps=num_time_slices)
evo.convert(evolved_hamiltonian)

# get the evolved state
evolved_state = evolved_hamiltonian @ initial_state

# evaluate the target observable at the evolved state
print((~StateFn(observable) @ evolved_state).eval())

(-0.6536436208636165+0j)
