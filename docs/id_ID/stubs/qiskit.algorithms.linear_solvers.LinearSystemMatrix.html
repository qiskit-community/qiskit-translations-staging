LinearSystemMatrix¶

class LinearSystemMatrix(num_state_qubits, tolerance, evolution_time, name='ls_matrix')[sumber]¶

Basis: qiskit.circuit.library.blueprintcircuit.BlueprintCircuit, abc.ABC

Base class for linear system matrices.

Parameter

  • num_state_qubits (int) -- the number of qubits where the unitary acts.

  • tolerance (float) -- the accuracy desired for the approximation

  • evolution_time (float) -- the time of the Hamiltonian simulation

  • name (str) -- The name of the object.

metadata¶

The user provided metadata associated with the circuit

The metadata for the circuit is a user provided dict of metadata for the circuit. It will not be used to influence the execution or operation of the circuit, but it is expected to be passed between all transforms of the circuit (ie transpilation) and that providers will associate any circuit metadata with the results it returns from execution of that circuit.
