ProteinFoldingProblem¶

class ProteinFoldingProblem(peptide, interaction, penalty_parameters)[source]¶

Bases: qiskit_nature.problems.sampling.sampling_problem.SamplingProblem

Defines a protein folding problem that can be passed to algorithms. Example initialization:

penalty_terms = PenaltyParameters(15, 15, 15)
main_chain_residue_seq = "SAASSASAAG"
side_chain_residue_sequences = ["", "", "A", "A", "A", "A", "A", "A", "S", ""]

peptide = Peptide(main_chain_residue_seq, side_chain_residue_sequences)

mj_interaction = MiyazawaJerniganInteraction()

protein_folding_problem = ProteinFoldingProblem(peptide, mj_interaction, penalty_terms)
qubit_op = protein_folding_problem.qubit_op()
Parameters

  • peptide (Peptide) -- A peptide object that defines the protein subject to the folding problem.

  • interaction (Interaction) -- A type of interaction between the beads of the peptide.

  • penalty_parameters (PenaltyParameters) -- Parameters that define the strength of constraints enforcing in the problem.

Attributes

unused_qubits

Returns the list of indices for qubits in the original problem formulation that were removed during compression.

Methods

interpret()

Interprets results of an optimization.

qubit_op()

Builds a qubit operator for the Hamiltonian encoding a protein folding problem.
