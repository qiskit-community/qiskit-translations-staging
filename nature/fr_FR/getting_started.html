Premiers Pas¶

Installation¶

Qiskit Nature depends on the main Qiskit package which has its own Qiskit Getting Started detailing the installation options for Qiskit and its supported environments/platforms. You should refer to that first. Then the information here can be followed which focuses on the additional installation specific to Qiskit Nature.

Certaines fonctions de Qiskit Nature ont été rendues facultatives lorsque le code ou le programme de support dépendants ne sont pas (ou ne peuvent pas être) installés par défaut. Celles-ci comprennent, par exemple, la bibliothèque classique/les programmes pour les problèmes moléculaires. Voir Installations optionnelles pour plus d’informations.

La façon la plus simple de commencer est de suivre le début de “Démarrer localement” pour Qiskit ici Qiskit Getting Started

Dans votre environnement virtuel où vous avez installé Qiskit ajoutez simplement nature à la liste supplémentaire de manière similaire à la façon dont le support supplémentaire de visualization est installé pour Qiskit, i.e :

pip install qiskit[nature]

Il est utile de signaler que si vous êtes un utilisateur de zsh (qui est le shell par défaut sur les nouvelles versions de macOS), vous aurez besoin de mettre qiskit[nature] entre guillemets :

pip install 'qiskit[nature]'

L’installation de Qiskit Nature à partir des sources vous permet d’accéder à la dernière version mise à jour en cours de développement au lieu d’utiliser la version dans le dépôt Python Package Index (PyPI). Cela vous permettra d’inspecter et d’étendre plus efficacement la dernière version du code de Qiskit Nature.

Étant donné que Qiskit Nature dépend de Qiskit, et que ses derniers changements peuvent nécessiter des fonctionnalités nouvelles ou modifiées de Qiskit, vous devriez d’abord suivre les instructions de Qiskit « Installer à partir de la source » ici Qiskit pour commencer

Installing Qiskit Nature from Source

En utilisant le même environnement de développement que celui où vous avez installé Qiskit, vous êtes prêt à installer Qiskit Nature.

  1. Cloner le dépôt de Qiskit Nature.

    git clone https://github.com/Qiskit/qiskit-nature.git

  2. Un répertoire local appelé qiskit-nature est créé automatiquement.

    cd qiskit-nature

  3. Si vous voulez exécuter des tests ou des vérifications de liaison, installez les prérequis du développeur.

    pip install -r requirements-dev.txt

  4. Installez qiskit-nature.

    pip install .

Si vous voulez l’installer en mode édition, ce qui signifie que les modifications de code du projet ne nécessitent pas de réinstallation pour être appliquées, vous pouvez le faire avec:

pip install -e .

Installations optionnelles¶

Qiskit Nature prend en charge l’utilisation de différentes bibliothèques et programmes classiques, par l’intermédiaire de pilotes, qui calculent des informations moléculaires, comme les intégrales d’un et deux corps. Ceci est nécessaire comme une entrée de problème pour les algorithmes qui calculent les propriétés des molécules, comme l’énergie de l’état fondamental, donc au moins une telle bibliothèque/programme devrait être installée. Comme vous pouvez choisir le pilote que vous utilisez, vous pouvez en installer autant que vous le souhaitez, qui sont supportés par votre plate-forme, etc.

See Driver installation which lists each driver and how to install the dependent library/program that it requires.
