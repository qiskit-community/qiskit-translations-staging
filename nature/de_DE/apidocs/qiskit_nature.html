Qiskit Nature API Reference¶

Qiskit Nature module (qiskit_nature)¶

Qiskit Nature provides function to experiment with quantum computing for natural science problems, such as in chemistry and physics. For example computing the ground state energy or excited state energies of molecules.

The top-level classes and submodules of qiskit_nature are:

QiskitNatureError

Class for errors returned by Qiskit's Nature module.

UnsupportMethodError

Class used if a driver doesn't support given method.
