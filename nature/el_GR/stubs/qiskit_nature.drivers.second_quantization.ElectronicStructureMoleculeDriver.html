ElectronicStructureMoleculeDriver¶

class ElectronicStructureMoleculeDriver(molecule, basis='sto3g', method=MethodType.RHF, driver_type=ElectronicStructureDriverType.AUTO, driver_kwargs=None)[πηγή]¶

Βάσεις: qiskit_nature.drivers.second_quantization.electronic_structure_driver.ElectronicStructureDriver

Molecule based electronic structure driver

Παράμετροι

  • molecule (Molecule) – molecule

  • basis (str) – basis set

  • method (MethodType) – The method type to be used for the calculation.

  • driver_type (ElectronicStructureDriverType) – type of driver to be used. If AUTO is selected, it will use the first driver installed and that supports the given method in the following order: PYSCF, PSI4, PYQUANTE, GAUSSIAN

  • driver_kwargs (Optional[Dict[str, Any]]) – kwargs to be passed to driver
