AdaptVQE¶

class AdaptVQE(qubit_converter, solver, threshold=1e-05, delta=1, max_iterations=None)[소스]¶

기반 클래스: qiskit_nature.algorithms.ground_state_solvers.ground_state_eigensolver.GroundStateEigensolver

A ground state calculation employing the AdaptVQE algorithm.

매개변수

  • qubit_converter (QubitConverter) – a class that converts second quantized operator to qubit operator

  • solver (MinimumEigensolverFactory) – a factory for the VQE solver employing a UCCSD ansatz.

  • threshold (float) – the energy convergence threshold. It has a minimum value of 1e-15.

  • delta (float) – the finite difference step size for the gradient computation. It has a minimum value of 1e-5.

  • max_iterations (Optional[int]) – the maximum number of iterations of the AdaptVQE algorithm.
