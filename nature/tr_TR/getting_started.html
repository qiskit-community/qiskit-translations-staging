Kurulum¶

Qiskit Nature; Qiskit için kurulum seçeneklerini ve onun desteklenen ortam/platformlarını detaylandıran Qiskit Başlarken adındaki kendi ana Qiskit paketine bağlıdır. Öncelikle buna bakmalısınız. Sonrasında, burada yer alan Qiskit Nature’a özgü ek kuruluma ilişkin bilgiler takip edilebilir.

Qiskit Nature, bağımlı kodun ve/veya destek program(lar)ının varsayılan olarak kurulmadığı (veya kurulamadığı) durumlarda isteğe bağlı hâle getirilmiş bazı işlevlere sahiptir. Bunlar, örneğin moleküler problemler için klasik kütüphaneleri/programları içerir. Daha fazla bilgi için opsiyonel_kurulumlar bölümüne bakınız.

Başlamanın en kolay yolu, Qiskit için ‘Yerel Olarak Başlatma’ bölümünü buradan takip ederek başlamaktır: Qiskit Başlarken

Qiskit’i kurduğunuz sanal ortamınızda, Qiskit için ekstra “görselleştirme” desteğinin nasıl kurulduğuna benzer şekilde, “nature”ı ekstra listeye eklemeniz yeterlidir, yani:

pip install qiskit[nature]

Bir zsh kullanıcısıysanız (macOS’un daha yeni sürümlerinde varsayılan kabuktur), ``qiskit[nature]``ı tırnak içine koymanız gerekeceğini belirtmekte fayda var:

pip install 'qiskit[nature]'

Qiskit Nature’ı kaynaktan yüklemek, Python Paket Dizini (PyPI) deposundaki sürümü kullanmak yerine, geliştirilmekte olan en son güncellenen sürüme erişmenizi sağlar. Bu size Qiskit Nature kodunun en son sürümünü daha verimli bir şekilde inceleme ve genişletme kabiliyeti verecektir.

Qiskit Nature, Qiskit’e bağlı olduğundan ve onun son değişiklikleri Qiakit’in yeni ya da değiştirilmiş özelliklerini gerektirebileceğinden, ilk olarak burayı takip etmeniz gerekmektedir: “Install from source” instructions here Qiskit Getting Started

Installing Qiskit Nature from Source

Qiskit’i kurduğunuz aynı geliştirme ortamını kullanarak Qiskit Nature’ı kurmaya hazırsınız.

  1. Qiskit Nature deposunu klonla.

    git clone https://github.com/Qiskit/qiskit-nature.git

  2. Depoyu klonlamak, qiskit-nature adlı yerel bir klasör oluşturur.

    cd qiskit-nature

  3. Test veya tiftikleme kontrolleri yapmak istiyorsanız, geliştirici gereksinimlerini yükleyin.

    pip install -r requirements-dev.txt

  4. qiskit-nature yükleyin.

    pip install .

Düzenlenebilir modda yüklemek istiyorsanız, yani projedeki kod değişikliklerinin yeniden yüklenmesi gerekmiyorsa, bunu şu şekilde yapabilirsiniz:

pip install -e .

Opsiyonel yüklemeler¶

Qiskit Nature, bir ve iki gövde integralleri gibi moleküler bilgileri hesaplayan sürücüler aracılığıyla farklı klasik kitaplıkların ve programların kullanımını destekler. Bu, temel durum enerjisi gibi moleküllerin özelliklerini hesaplayan algoritmalar için problem girdisi olarak gereklidir, bu nedenle bu tür en az bir kitaplık/program kurulmalıdır. Hangi sürücüyü kullanacağınızı seçebildiğiniz gibi, platformunuz tarafından desteklenen vb. istediğiniz kadar çok veya az yükleyebilirsiniz.

See Driver installation which lists each driver and how to install the dependent library/program that it requires.
