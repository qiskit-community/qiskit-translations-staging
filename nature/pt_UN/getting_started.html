Primeiros passos¶

Instalação¶

O Qiskit Nature depende do pacote Qiskit principal que tem seu próprio Qiskit Getting Started detalhando as opções de instalação para Qiskit e seus ambientes/plataformas suportados. Você deve se referir a isso primeiro. Em seguida, as informações aqui podem ser seguidas, as quais se concentram na instalação adicional específica para Qiskit Nature.

O Qiskit Nature tem algumas funções que se tornaram opcionais onde o código dependente e/ou programa(s) de suporte não são (ou não podem ser) instalados por padrão. Estes incluem, por exemplo, biblioteca/programas clássicos para problemas moleculares. Veja Instalações opcionais para mais informações.

A maneira mais simples de começar é seguir o guia “Iniciar localmente” para o Qiskit aqui Qiskit Getting Started

Em seu ambiente virtual onde você instalou o Qiskit, simplesmente adicione nature à lista extra de maneira semelhante a como o suporte extra visualization é instalado para o Qiskit, ou seja:

pip install qiskit[nature]

Vale a pena ressaltar que se você for um usuário zsh (que é o shell padrão nas versões mais recentes do macOS), você precisará usar qiskit[nature] entre aspas:

pip install 'qiskit[nature]'

A instalação do Qiskit Nature da fonte permite que você acesse a versão atualizada mais recentemente em desenvolvimento, em vez de usar a versão no repositório Python Package Index (PyPI). Isso lhe dará a capacidade de inspecionar e estender a versão mais recente do código Qiskit Nature com mais eficiência.

Uma vez que o Qiskit Nature depende do Qiskit, e suas últimas mudanças podem exigir recursos novos ou modificados do Qiskit, você deve primeiro seguir as instruções «Install from source» do Qiskit aqui Qiskit Getting Started

Installing Qiskit Nature from Source

Usando o mesmo ambiente de desenvolvimento em que instalou o Qiskit, você está pronto para instalar o Qiskit Nature.

  1. Clone o repositório Qiskit Nature.

    git clone https://github.com/Qiskit/qiskit-nature.git

  2. A clonagem do repositório cria uma pasta local chamada qiskit-nature.

    cd qiskit-nature

  3. Se você quiser executar testes ou análise de código, instale os requisitos de desenvolvedor.

    pip install -r requirements-dev.txt

  4. Instale qiskit-nature.

    pip install .

Se você quiser instalá-lo em modo editável, o que significa que as alterações de código no projeto não requerem que uma reinstalação seja aplicada, você pode fazer isso com:

pip install -e .

Instalações opcionais¶

O Qiskit Nature suporta o uso de diferentes bibliotecas e programas clássicos, por meio de drivers, que calculam informação molecular, como integrais de um e dois corpos. Isso é necessário como entrada do problema para algoritmos que calculam propriedades de moléculas, como a energia do estado fundamental, logo pelo menos uma biblioteca/programa deve ser instalada. Como você pode escolher qual driver utilizar, você pode instalar quantos desejar, que sejam suportados por sua plataforma etc.

Veja Instalação de driver que lista cada driver e como instalar a biblioteca/programa dependente que ele necessita.
