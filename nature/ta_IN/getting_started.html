தொடங்குதல்¶

நிறுவல்¶

Qiskit நேச்சர் அதன் சொந்த Qiskit Getting Started மற்றும் அதன் ஆதரவு சூழல்கள் / தளங்களுக்கான நிறுவல் விருப்பங்களை விவரிக்கும் முக்கிய qiskit தொகுப்பைப் பொறுத்தது. நீங்கள் அதை முதலில் குறிப்பிட வேண்டும். Qiskit நேச்சருக்கு குறிப்பிட்ட கூடுதல் நிறுவலை மையமாகக் கொண்ட தகவல்களை இங்கே பின்பற்றலாம்.

Qiskit நேச்சர் சில செயல்பாடுகளைக் கொண்டுள்ளது, அவை கட்டாயமற்றவையாக அமைக்கப்பட்டுள்ளது அங்கு சார்பு குறியீடு மற்றும் / அல்லது ஆதரவு நிரல் (கள்) இயல்பாக நிறுவப்படவில்லை (அல்லது இருக்க முடியாது). எடுத்துக்காட்டாக, மூலக்கூறு சிக்கல்களுக்கான கிளாசிக்கல் நூலகம் / நிரல்கள் இதில் அடங்கும். மேலும் தகவலுக்கு விருப்ப நிறுவல்கள் ஐப் பார்க்கவும்.

Qiskit தொடங்குவதற்கான எளிய வழி இங்கே 'Start locally' என்பதைப் பின்பற்றுவது Qiskit Getting Started

நீங்கள் qiskit-டை நிறுவிய உங்கள் மெய்நிகர் சூழலில்,qiskit-டிற்கு கூடுதல் visualization ஆதரவு எவ்வாறு நிறுவப்பட்டுள்ளது என்பதற்கு ஒத்த வகையில் கூடுதல் பட்டியலில் nature ஐச் சேர்க்கவும், அதாவது:

pip install qiskit[nature]

நீங்கள் ஒரு zsh பயனராக இருந்தால் (இது மேக்ஓஸின் புதிய பதிப்புகளில் இயல்புநிலை ஷெல்), நீங்கள் மேற்கோள்களில் qiskit[nature] வைக்க வேண்டும் என்பதை சுட்டிக்காட்ட வேண்டியது அவசியம்:

pip install 'qiskit[nature]'

ஆரம்ப இடத்திலிருந்து Qiskit நேச்சரை நிறுவுவது பைத்தான் தொகுப்பு அட்டவணை (பைபிஐ) களஞ்சியத்தில் பதிப்பைப் பயன்படுத்துவதற்குப் பதிலாக, சமீபத்தில் புதுப்பிக்கப்பட்ட பதிப்பை வளர்ச்சியின் கீழ் அணுக அனுமதிக்கிறது. இது Qiskit நேச்சர் குறியீட்டின் சமீபத்திய பதிப்பை மிகவும் திறமையாக ஆய்வு செய்து நீட்டிக்கும் திறனை உங்களுக்கு வழங்கும்.

Qiskit நேச்சர் கிஸ்கிட்டைப் பொறுத்தது என்பதால், அதன் சமீபத்திய மாற்றங்களுக்கு Qiskit-டின் புதிய அல்லது மாற்றப்பட்ட அம்சங்கள் தேவைப்படலாம் என்பதால், நீங்கள் முதலில் Qiskit-டின் "Install from source" இங்கே வழிமுறைகளைப் பின்பற்ற வேண்டும் Qiskit Getting Started

Installing Qiskit Nature from Source

நீங்கள் Qiskit-டை நிறுவிய அதே வளர்ச்சி சூழலைப் பயன்படுத்தி Qiskit நேச்சரை நிறுவ தயாராக உள்ளீர்கள்.

  1. Qiskit நேச்சர் களஞ்சியத்தை குளோன் செய்யுங்கள்.

    git clone https://github.com/Qiskit/qiskit-nature.git

  2. களஞ்சியத்தை குளோன் செய்வது qiskit-nature என்ற அந்த இடத்திற்குறிய கோப்புறையை உருவாக்குகிறது.

    cd qiskit-nature

  3. நீங்கள் சோதனைகள் அல்லது மெல்லிய சரிபார்ப்புகளை இயக்க விரும்பினால், டெவலப்பர் தேவைகளை நிறுவவும்.

    pip install -r requirements-dev.txt

  4. qiskit-nature ஐ நிறுவவும்.

    pip install .

நீங்கள் இதை திருத்தக்கூடிய முறையில் நிறுவ விரும்பினால், திட்டத்தின் குறியீடு மாற்றங்களுக்கு மீண்டும் நிறுவல் தேவையில்லை என்று பொருள், இதை நீங்கள் செய்யலாம்:

pip install -e .

விருப்ப நிறுவல்கள்¶

ஒன்று மற்றும் இரண்டு உடல் ஒருங்கிணைப்புகள் போன்ற மூலக்கூறு தகவல்களைக் கணக்கிடும் இயக்கிகள் வழியாக Qiskit நேச்சர் வெவ்வேறு கிளாசிக்கல் நூலகங்கள் மற்றும் நிரல்களைப் பயன்படுத்துவதை ஆதரிக்கிறது. தரை நிலை ஆற்றல் போன்ற மூலக்கூறுகளின் பண்புகளை கணக்கிடும் வழிமுறைகளுக்கு சிக்கல் உள்ளீடாக இது தேவைப்படுகிறது, எனவே குறைந்தபட்சம் இதுபோன்ற ஒரு நூலகம் / நிரல் நிறுவப்பட வேண்டும். நீங்கள் எந்த இயக்கியைப் பயன்படுத்துகிறீர்கள் என்பதைத் தேர்வுசெய்யும்போது, ​​உங்கள் இயங்குதளத்தால் ஆதரிக்கப்படும் பலவற்றை அல்லது நீங்கள் விரும்பும் சிலவற்றை நிறுவலாம்.

Driver installation இது ஒவ்வொரு டிரைவரையும் பட்டியலிடுகிறது மற்றும் அது தேவைப்படும் சார்பு நூலகம்/நிரலை எவ்வாறு நிறுவுவதை பார்க்கவும்.
