தொடங்குதல்¶

நிறுவல்¶

Qiskit உகப்பாக்கம் அதன் சொந்த Qiskit Getting Started Qiskit மற்றும் அதன் ஆதரவு சூழல்கள் / தளங்களுக்கான நிறுவல் விருப்பங்களை விவரிக்கும் முக்கிய கிஸ்கிட் தொகுப்பைப் பொறுத்தது. நீங்கள் அதை முதலில் குறிப்பிட வேண்டும். Qiskit உகப்பாக்கத்திற்கு குறிப்பிட்ட கூடுதல் நிறுவலில் கவனம் செலுத்தும் தகவல்களை இங்கே பின்பற்றலாம்.

Qiskit ஆப்டிமைசேஷனில் சில செயல்பாடுகள் விருப்பத்தேர்வு செய்யப்பட்டுள்ளன, அங்கு சார்ந்த குறியீடு மற்றும்/அல்லது ஆதரவு நிரல் (கள்) இயல்பாக நிறுவப்படவில்லை (அல்லது இருக்க முடியாது). அவை IBM CPLEX, CVXPY மற்றும் Matplotlib. மேலும் தகவலுக்கு :ref: optional_installs ஐப் பார்க்கவும்.

தொடங்குவதற்கான எளிய வழி இங்கே கிஸ்கிட்டிற்கான 'Start locally' என்பதைப் பின்பற்றுவது Qiskit Getting Started

நீங்கள் Qiskitடை நிறுவிய உங்கள் மெய்நிகர் சூழலில், Qiskitற்கு கூடுதல் visualization ஆதரவு எவ்வாறு நிறுவப்பட்டுள்ளது என்பதைப் போலவே கூடுதல் பட்டியலில் optimization ஐச் சேர்க்கவும், அதாவது:

pip install qiskit[optimization]

நீங்கள் ஒரு zsh பயனராக இருந்தால் (இது macOS-ன் புதிய பதிப்புகளில் இயல்புநிலை ஷெல்), நீங்கள் மேற்கோள்களில் qiskit[optimization] வைக்க வேண்டும் என்பதை சுட்டிக்காட்ட வேண்டியது அவசியம்:

pip install 'qiskit[optimization]'

மூலத்திலிருந்து Qiskit உகப்பாக்கத்தை நிறுவுவது பைதான் தொகுப்பு அட்டவணை (PyPI) களஞ்சியத்தில் பதிப்பைப் பயன்படுத்துவதற்குப் பதிலாக, சமீபத்தில் புதுப்பிக்கப்பட்ட பதிப்பை வளர்ச்சியின் கீழ் அணுக அனுமதிக்கிறது. இது qiskit உகப்பாக்கம் குறியீட்டின் சமீபத்திய பதிப்பை மிகவும் திறமையாக ஆய்வு செய்து நீட்டிக்கும் திறனை உங்களுக்கு வழங்கும்.

Qiskit உகப்பாக்கம் qiskitடைப் பொறுத்தது என்பதால், அதன் சமீபத்திய மாற்றங்களுக்கு qiskitடின் புதிய அல்லது மாற்றப்பட்ட அம்சங்கள் தேவைப்படலாம் என்பதால், நீங்கள் முதலில் கிஸ்கிட்டின் "Install from source" இங்கே வழிமுறைகளைப் பின்பற்ற வேண்டும் Qiskit Getting Started

Installing Qiskit Optimization from Source

நீங்கள் Qiskitடை நிறுவிய அதே வளர்ச்சி சூழலைப் பயன்படுத்தி கிஸ்கிட் உகப்பாக்கலை நிறுவ தயாராக உள்ளீர்கள்.

  1. Qiskit உகப்பாக்கம் களஞ்சியத்தை குளோன் செய்யுங்கள்.

    git clone https://github.com/Qiskit/qiskit-optimization.git

  2. களஞ்சியத்தை குளோன் செய்வது qiskit-optimization எனப்படும் உள்ளூர் கோப்புறையை உருவாக்குகிறது.

    cd qiskit-optimization

  3. நீங்கள் சோதனைகள் அல்லது லிண்டிங் காசோலைகளை இயக்க விரும்பினால், டெவலப்பர் தேவைகளை நிறுவவும்.

    pip install -r requirements-dev.txt

  4. qiskit-optimization ஐ நிறுவவும்.

    pip install .

நீங்கள் அதை திருத்தக்கூடிய முறையில் நிறுவ விரும்பினால், அதாவது திட்டத்தில் குறியீடு மாற்றங்களை மீண்டும் நிறுவுதல் தேவையில்லை, நீங்கள் இதைச் செய்யலாம்:

pip install -e .

விருப்ப நிறுவல்கள்¶

  • IBM CPLEX பயன்படுத்தி நிறுவப்படலாம் pip install 'qiskit-optimization[cplex]' வாசிப்பை செயல்படுத்த LP கோப்புகள் மற்றும் அதன் பயன்பாடு CplexOptimizer, க்கான மடக்கு cplex.Cplex. தற்போது இல்லை Python 3.9 பதிப்பு இன் CPLEX. இந்த வழக்கில், CPLEX நிறுவல் கட்டளை எந்த விளைவையும் ஏற்படுத்தாது.

  • ** CVXPY **, தொகுப்பை நிறுவ pip install 'qiskit-optimization [cvx]' கட்டளையைப் பயன்படுத்தி நிறுவப்படலாம். CVXPY நிறுவப்பட்டிருப்பது கோமன்ஸ்-வில்லியம்சன் வழிமுறையை உகப்பாக்கியாக GoemansWilliamsonOptimizer பயன்படுத்த உதவும்.

  • Matplotlib, தொகுப்பை நிறுவ pip install 'qiskit-optimization [matplotlib]' கட்டளையைப் பயன்படுத்தி நிறுவப்படலாம். மேட்லோட்லிப் நிறுவப்படுவது வரைபட தேர்வுமுறை பயன்பாட்டு வகுப்புகளில் draw முறையைப் பயன்படுத்த உதவும்.

  • Gurobipy தொகுப்பை நிறுவ pip install 'qiskit-optimization[gurobi]' கட்டளையைப் பயன்படுத்தி நிறுவப்படலாம். குரோபிபி நிறுவப்படுவது குரோபிஆப்டிமைசரைப் பயன்படுத்த உதவும்.
