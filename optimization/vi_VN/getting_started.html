Bắt đầu¶

Cài đặt¶

Qiskit Optimization phụ thuộc vào gói Qiskit chính sở hữu ' Qiskit Getting Started <https://qiskit.org/documentation/getting_started.html>` __ riêng, trình bày các lựa chọn cài đặt cho Qiskit và các môi trường/nền tảng được hỗ trợ của nó. Bạn nên tham khảo mục đó trước. Tiếp đó, các thông tin tập trung vào những cài đặt bổ sung cụ thể cho Qiskit Optimization có thể được xem thêm tại đây.

Qiskit Optimization có một số chức năng đã được điều chỉnh thành tùy chọn để các code phụ thuộc và/hoặc (các) chương trình hỗ trợ không (hoặc không thể) được cài đặt theo mặc định. Các chức năng đó là IBM CPLEX, CVXPY và Matplotlib. Xem Các cài đặt tùy chọn để biết thêm thông tin.

Cách dễ nhất để bắt đầu là làm theo hướng dẫn mở đầu của "Bắt đầu cục bộ" dành cho Qiskit tại Qiskit Getting Started

Trong môi trường ảo mà bạn đã cài đặt Qiskit, chỉ cần thêm optimization vào danh sách bổ sung tương tự như cách cài đặt hỗ trợ bổ sung visualization cho Qiskit, nghĩa là:

pip install qiskit[optimization]

It is worth pointing out that if you're a zsh user (which is the default shell on newer versions of macOS), you'll need to put qiskit[optimization] in quotes:

pip install 'qiskit[optimization]'

Cài đặt Qiskit Optimization từ nguồn cho phép bạn truy cập phiên bản đang phát triển được cập nhật gần nhất thay vì sử dụng phiên bản trong kho lưu trữ Python Package Index (PyPI). Điều này sẽ giúp bạn kiểm tra và mở rộng code của phiên bản Qiskit Optimization mới nhất một cách hiệu quả hơn.

Do Qiskit Optimization phụ thuộc vào Qiskit, và những thay đổi mới nhất của nó có thể sẽ yêu cầu các tính năng mới hoặc chỉnh sửa các tính năng của Qiskit, nên trước tiên bạn cần làm theo hướng dẫn "Install from source" của Qiskit tại Qiskit Getting Started

Installing Qiskit Optimization from Source

Bây giờ bạn đã sẵn sàng để cài đặt Qiskit Optimization trên cùng môi trường phát triển mà bạn đã cài đặt Qiskit trước đó.

  1. Nhân bản kho lưu trữ của Qiskit Optimization.

    git clone https://github.com/Qiskit/qiskit-optimization.git

  2. Nhân bản kho lưu trữ sẽ tạo ra một thư mục cục bộ có tên qiskit-optimization.

    cd qiskit-optimization

  3. Nếu bạn muốn chạy test hoặc linting, hãy cài đặt các yêu cầu dành cho nhà phát triển.

    pip install -r requirements-dev.txt

  4. Cài đặt qiskit-optimization.

    pip install .

Nếu bạn muốn cài đặt ở chế độ có thể chỉnh sửa được, nghĩa là khi bạn thay đổi code trong project thì bạn không cần phải cài đặt lại, làm điều này bằng cách:

pip install -e .

Các cài đặt tùy chọn¶

  • IBM CPLEX may be installed using pip install 'qiskit-optimization[cplex]' to enable the reading of LP files and the usage of the CplexOptimizer, wrapper for cplex.Cplex. Currently there is no python 3.9 version of CPLEX. In this case, the CPLEX install command will have no effect.

  • * CVXPY, có thể được cài đặt bằng lệnh pip install 'qiskit-optimization[cvx]' để cài đặt gói. CVXPY được cài đặt sẽ cho phép sử dụng thuật toán Goemans-Williamson như một trình tối ưu hóa GoemansWilliamsonOptimizer.

  • ** Matplotlib**, có thể được cài đặt bằng lệnh pip install 'qiskit-optimization[matplotlib]' để cài đặt gói. Matplotlib được cài đặt sẽ cho phép sử dụng phương pháp vẽ draw trong các lớp ứng dụng tối ưu đồ thị.

  • Gurobipy may be installed using command pip install 'qiskit-optimization[gurobi]' to install the package. Gurobipy being installed will enable the usage of the GurobiOptimizer.
