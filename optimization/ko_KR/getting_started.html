시작하기¶

설치¶

Qiskit Optimization은 Qiskit및 지원되는 환경/플랫폼의 설치 옵션을 자세히 설명하는 Qiskit Getting Started 를 가지고 있는 기본 Qiskit 패키지에 의존한다. 먼저 그것을 참고해야 한다. 그 후 Qiskit Optimization의 추가 설치에 초점을 맞추는 이 내용을 따라라.

Qiskit Optimization에는 종속 코드 및/또는 지원 프로그램이 기본적으로 설치되어 있지 않은 (또는 설치될 수 없는) 일부 선택적인 기능들이 있다. 이에는 IBM CPLEX, CVXPY 및 Matplotlib가 있다. 자세한 내용은 선택적 설치 를 참조하여라.

시작할 수 있는 가장 간단한 방법은 Qiskit 에서 〈로컬로 시작하기〉 `Qiskit Getting Started <https://qiskit.org/documentation/getting_started.html>`__를 따르는 것이다.

Qiskit을 설치한 가상 환경에서 추가적인 visualization 지원을 설치한 방법과 마찬가지로 optimization 를 단순히 추가 리스트에 추가한다.

pip install qiskit[optimization]

Zsh 사용자 (최신 버전의 macOS 기본 셸) 인 경우 qiskit[optimization] 을 따옴표로 묶어야 한다:

pip install 'qiskit[optimization]'

소스 코드로부터 Qiskit Optimization을 설치하는 것은 Python Package Index (PyPI) repository로 부터 설치하는 것 대신에 가장 최신에 업데이트된 버전을 사용하도록 해준다. 이를 통해 손쉽게 Qiskit Optimization의 최신 코드를 들여다 보고 확장할 수 있다.

Qiskit Optimization은 Qiskit에 의존하므로 최신 변경 사항에는 Qiskit의 새로운 기능이나 변경된 기능이 필요할 수 있다. 따라서 다음 문서에 있는 Qiskit의 《Install from source》 의 지시사항을 먼저 수행해야 한다: Qiskit Getting Started

Installing Qiskit Optimization from Source

Qiskit을 설치한 동일한 개발 환경을 사용하여 Qiskit Optimization을 설치할 준비가 되었다.

  1. Qiskit Optimization repository를 복제하라.

    git clone https://github.com/Qiskit/qiskit-optimization.git

  2. Repository 복제는 qiskit-optimization 라는 이름의 로컬 폴더를 생성한다.

    cd qiskit-optimization

  3. 만약 테스트 코드를 실행하거나 철자법 검사를 하고자 한다면, 개발자 요구사항 라이브러리를 설치하라.

    pip install -r requirements-dev.txt

  4. qiskit-optimization 를 설치하라.

    pip install .

만약, 프로젝트에 코드가 변화하였을 때 재설치가 필요하지 않는, 수정 가능한 모드로 설치하고자 한다면, 다음의 명령어를 사용하라.

pip install -e .

선택적 설치¶

  • IBM CPLEX 는 LP 파일을 읽고 cplex.Cplex 의 래퍼인 CplexOptimizer 을 사용하기 위해 pip install 〈qiskit-optimization[cplex]〉 명령어로 패키지를 설치할 수 있다. 현재 python 3.9 버전용 CPLEX는 아직 없기 때문에 해당 버전의 경우 CPLEX 설치 명령은 동작하지 않는다.

  • CVXPY 는 pip install 〈qiskit-optimization[cvx]〉 커맨드로 패키지를 설치할 수 있다. CVXPY는 Goemans-Williamson 알고리즘을 GoemansWilliamsonOptimizer 옵티마이저로 사용할 수 있게 한다.

  • Matplotlib 는 pip install 〈qiskit-optimization[matplotlib]〉 커맨드로 패키지를 설치할 수 있다. Matplotlib은 그래프 최적화 응용 (graph optimization application) 클래스에 있는 draw 메서드를 사용할 수 있게 한다.

  • Gurobipy 는 pip install 〈qiskit-optimization[gurobi]〉 명령어를 사용하여 설치할 수 있다. Gurobipy를 설치하면 GurobiOptimizer를 사용할 수 있다.
