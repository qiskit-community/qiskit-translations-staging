from qiskit import *

繪製直方圖¶

為了圖示化量子電路在實體量子電腦或 qasm_simulator 執行的計算結果，我們提供一個簡單的函數做使用

plot_histogram(data)

我們以兩個量子位元的 Bell state 做舉例

[9]:

# quantum circuit to make a Bell state
bell = QuantumCircuit(2, 2)
bell.h(0)
bell.cx(0, 1)

meas = QuantumCircuit(2, 2)
meas.measure([0,1], [0,1])

# execute the quantum circuit
backend = BasicAer.get_backend('qasm_simulator') # the device to run on
circ = bell.compose(meas)
result = backend.run(transpile(circ, backend), shots=1000).result()
counts  = result.get_counts(circ)
print(counts)

{'11': 480, '00': 520}

[10]:

plot_histogram(counts)

[10]:

繪製直方圖函數的可調參數¶

plot_histogram() 有幾個選項可調整輸出圖片。第一個選項是 legend，可用來為執行結果添加標籤。它的輸入是一系列用於標示每個執行結果的字串。它在同一個直方圖中繪製多個執行結果時十分好用。sort 可用來調整直方圖中的長條的呈現順序。它可以用 asc 設為升序或 desc 降序。number_to_keep 可輸入一個整數表示需要顯示的項目數量。其餘的則會在一個稱為 rest 的單一條欄中集中顯示。您可以使用 color 來調整長條的顏色。它需要輸入一個或一組字串，以用於設置每一個執行結果的長條顔色。 您可以用 bar_labels 選擇標籤是否顯示在長條上方。最後一個可以設置選項是 figsize。它需要輸入一個以英寸為單位的元組作爲圖片尺寸，來製作輸出圖。

[12]:

# Execute 2-qubit Bell state again
second_result = backend.run(transpile(circ, backend), shots=1000).result()
second_counts  = second_result.get_counts(circ)
# Plot results with legend
legend = ['First execution', 'Second execution']
plot_histogram([counts, second_counts], legend=legend)

使用 plot_histogram() 的輸出¶

當使用 plot_histogram() 函數時，它會返回一個 matplotlib.Figure 來呈現視覺化。Jupyter notebooks 知道該返回類型，並在本教程中為我們所用。但在 Jupyter 之外執行時，您無法自動擁有此功能。不過，matplotlib.Figure 類擁有原生的視覺化顯示與存儲方法。您可以使用 plot_histogram() 返回對象中的 .show() ，以在新視窗中開啟圖片（假設您配置的 matplotlib 後端是可互動的）。或者，您也可以使用 .savefig('out.png')，將圖片儲存至 out.png。savefig() 方法需要輸入一個路徑，因此您可以自定義存儲地址與文檔名。

繪製狀態¶

在多數情況下，您可能是爲了代碼除錯，會想看到量子計算機中的態。 我們假設你有這個量子態（無論是從模擬或是 state tomography 中得到），而我們希望將這量子態視覺化。這需要指數級的運算資源，因此我們建議只檢視小量子系統的狀態。有幾種不同的函數可用於實現量子態的視覺化

plot_state_city(quantum_state)
plot_state_qsphere(quantum_state)
plot_state_paulivec(quantum_state)
plot_state_hinton(quantum_state)
plot_bloch_multivector(quantum_state)

量子態由一個狀態矩陣 \(\rho\) （Hermitian 矩陣）或狀態向量 \(|\psi\rangle\) （複數向量）表示。狀態矩陣與狀態向量的關係是：

\[\rho = |\psi\rangle\langle \psi|,\]

另有可以代表混合狀態（狀態向量的正數和）的更一般的形式：

\[\rho = \sum_k p_k |\psi_k\rangle\langle \psi_k |.\]

實現視覺化的函數有：

  • 'plot_state_city'：量子狀態的標準視圖，量子態矩陣的實部與虛部（imag）將繪製成像城市一樣。

  • 'plot_state_qsphere'：Qiskit 獨特的量子態視圖。它將狀態向量的振幅和相位繪製在一個球體中。振幅由箭頭的厚度表示，而相位則是由顏色。對於混合狀態，它將對每個部分顯示不同的 'qsphere'。

  • 'plot_state_paulivec'：以 Pauli 運算子作爲基底表示的量子態矩陣 \(\rho=\sum_{q=0}^{d^2-1}p_jP_j/d\)。

  • 'plot_state_hinton'：與 'city' 相同，但各部分面積的大小表示矩陣元素的值。

  • 'plot_bloch_multivector'：將量子狀態投影到單量子位元的空間，並繪製在 Bloch 球上。

量子態繪圖函數的選項¶

各種量子態繪圖函數提供了許多選項來調整繪圖的呈現方式。可用的選項取決於所使用的繪製函數。

plot_state_city() 選項

  • title (str)：表示繪圖標題的字串

  • figsize (tuple)：圖片尺寸（英吋）（寬度，高度）。

  • color (list)：len=2 表示矩陣元素的實部和虛部顏色的列表。

[22]:

plot_state_city(psi, title="My City", color=['black', 'orange'])

[22]:

plot_state_hinton() 選項

  • title (str)：表示繪圖標題的字串

  • figsize (tuple)：圖片尺寸（英吋）（寬度，高度）。

[23]:

plot_state_hinton(psi, title="My Hinton")

使用量子態繪圖函數的輸出¶

當使用任何量子態繪圖函數時，它會返回一個 matplotlib.Figure 來呈現視覺化。Jupyter notebooks 知道該返回類型，並在本教程中為我們所用。但在 Jupyter 之外執行時，您無法自動擁有此功能。不過，matplotlib.Figure 類擁有原生的視覺化顯示與存儲方法。您可以使用 plot_histogram() 返回對象中的 .show() ，以在新視窗中開啟圖片（假設您配置的 matplotlib 後端是可互動的）。或者，您也可以使用 .savefig('out.png')，將圖片儲存至 out.png。savefig() 方法需要輸入一個路徑，因此您可以自定義存儲地址與文檔名。

繪製 Bloch 向量圖¶

一個繪製量子系統的標準方法是使用 Bloch 向量。 這只適用於單一量子位元 ，並以 Bloch 向量作爲輸入。

Bloch 向量的定義為 \([x = \mathrm{Tr}[X \rho], y = \mathrm{Tr}[Y \rho], z = \mathrm{Tr}[Z \rho]]\)，其中 \(X\)，\(Y\) 和 \(Z\) 是單量子位元的 Pauli 運算子，而 \(\rho\) 是量子態矩陣。

[26]:

from qiskit.visualization import plot_bloch_vector

[27]:

plot_bloch_vector([0,1,0])

  • figsize (tuple)：圖片尺寸（英吋）（寬度，高度）。

plot_bloch_vector([0,1,0], title='My Bloch Sphere')
