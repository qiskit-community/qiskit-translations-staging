Volume Quântico¶

Introdução¶

Quantum Volume (QV) é um método para verificar o desempenho do dispositivo e uma métrica para quantificar o poder computacional de um dispositivo quântico. O método baseia-se no artigo “Validando computadores quânticos usando circuitos de modelos randomizados” (https://arxiv.org/abs/1811.12926).

Este notebook dá um exemplo para como usar o módulo ignis.verification.quantum_volume. Este exemplo específico mostra como executar circuitos de volume quântico até a profundidade 6, e os executará usando o simulador Aer com ruídos.

[1]:

#Import general libraries (needed for functions)
import numpy as np
import matplotlib.pyplot as plt
from IPython import display

#Import Qiskit classes classes
import qiskit
from qiskit.providers.aer.noise import NoiseModel
from qiskit.providers.aer.noise.errors.standard_errors import depolarizing_error, thermal_relaxation_error

#Import the qv function.
import qiskit.ignis.verification.quantum_volume as qv

Selecione os Parâmetros da Execução QV¶

Neste exemplo temos 6 qubits Q0, Q1, Q3, Q5, Q7, Q10. Vamos olhar para subconjuntos até o conjunto completo.

[2]:

#Qubit list
qubit_lists = [[0,1,3],[0,1,3,5],[0,1,3,5,7],[0,1,3,5,7,10]]
ntrials = 50

Geramos as sequências de volume quântico. Começamos com um pequeno exemplo (para que não leve muito tempo para executar).

qv_circs, qv_circs_nomeas = qv.qv_circuits(qubit_lists, ntrials)

Simular os circuitos ideais¶

O método de volume quântico requer que saibamos a saída ideal para cada circuito, portanto use o simulador de statevector em Aer para obter o resultado ideal.

[6]:

#The Unitary is an identity (with a global phase)
backend = qiskit.Aer.get_backend('statevector_simulator')
ideal_results = []
for trial in range(ntrials):
    print('Simulating trial %d'%trial)
    ideal_results.append(qiskit.execute(qv_circs_nomeas[trial], backend=backend, optimization_level=0).result())

Definir o modelo de ruído¶

Definimos um modelo de ruído para o simulador. Para simular o decaimento, adicionamos probabilidades de erro de despolarizamento aos portões de CNOT e U.

[8]:

noise_model = NoiseModel()
p1Q = 0.002
p2Q = 0.02
noise_model.add_all_qubit_quantum_error(depolarizing_error(p1Q, 1), 'u2')
noise_model.add_all_qubit_quantum_error(depolarizing_error(2*p1Q, 1), 'u3')
noise_model.add_all_qubit_quantum_error(depolarizing_error(p2Q, 2), 'cx')
#noise_model = None

Podemos executar as sequências de QV usando um Qiskit Aer Simulator (com algum modelo de ruído) ou usando um provedor IBMQ, e obter uma lista de resultados, result_list.

backend = qiskit.Aer.get_backend('qasm_simulator')
basis_gates = ['u1','u2','u3','cx'] # use U,CX for now
shots = 1024
exp_results = []
for trial in range(ntrials):
    print('Running trial %d'%trial)
    exp_results.append(qiskit.execute(qv_circs[trial], basis_gates=basis_gates, backend=backend, noise_model=noise_model, backend_options={'max_parallel_experiments': 0}).result())

Volume Quântico¶

Listar estatísticas para cada profundidade. Para cada lista de profundidade, se a profundidade foi bem-sucedida ou não, e com o que o intervalo de confiança. Para que uma profundidade seja bem sucedida, o intervalo de confiança deve ser> 97,5%.

[12]:

qv_success_list = qv_fitter.qv_success()
qv_list = qv_fitter.ydata
for qidx, qubit_list in enumerate(qubit_lists):
    if qv_list[0][qidx]>2/3:
        if qv_success_list[qidx][0]:
            print("Width/depth %d greater than 2/3 (%f) with confidence %f (successful). Quantum volume %d"%
                  (len(qubit_list),qv_list[0][qidx],qv_success_list[qidx][1],qv_fitter.quantum_volume()[qidx]))
        else:
            print("Width/depth %d greater than 2/3 (%f) with confidence %f (unsuccessful)."%
                  (len(qubit_list),qv_list[0][qidx],qv_success_list[qidx][1]))
    else:
        print("Width/depth %d less than 2/3 (unsuccessful)."%len(qubit_list))
